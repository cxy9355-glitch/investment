# 网易投资价值量化指标清单

更新时间：2026-03-28

## 1. 使用原则

这份清单不是为了把网易做成一张“大而全 peer table”，而是为了回答三个核心问题：

1. 网易现有现金流的质量到底有多高。
2. 网易未来 2-3 年的现金流可持续性到底有多强。
3. 当前估值是否已经过度反映了平台折价，却没有充分计入长青游戏、净现金和股东回报的价值。

对照对象分三层：

- 主锚：腾讯
- 国际运营锚：NEXON
- 辅助样本：三七互娱、吉比特、完美世界、恺英网络、心动、游族

## 2. 一级指标：直接决定投资价值判断

| 指标 | 定义 | 为什么重要 | 主对照对象 |
| --- | --- | --- | --- |
| 游戏业务收入占比 | 游戏及相关增值服务收入 / 总收入 | 判断网易是不是本质上的高质量游戏资产 | 腾讯 |
| 游戏业务营业利润率 | 游戏业务利润 / 游戏业务收入 | 体现核心主业的赚钱能力 | 腾讯、NEXON |
| 自由现金流率 | 自由现金流 / 总收入 | 比净利润更能反映真实现金创造能力 | 腾讯、NEXON |
| FCF / Non-GAAP 净利润 | 自由现金流 / Non-GAAP 净利润 | 判断利润是否高质量兑现为现金 | 腾讯、NEXON |
| 净现金 / 市值 | 净现金与短投净额 / 市值 | 衡量估值下行保护 | 腾讯、NEXON |
| 分红+回购 / FCF | 分红和回购总额 / 自由现金流 | 衡量资本回报纪律 | 腾讯、NEXON |
| 长青产品收入占比 | 核心老游戏收入 / 游戏业务收入 | 判断现有现金流护城河 | 腾讯、NEXON |
| 老游戏收入衰减速度 | 核心老游戏年流水或收入同比 | 判断资产寿命，而不是只看当前利润 | 腾讯、NEXON |
| 新游兑现率 | 上线新品中 12 个月后仍有核心流水贡献的比例 | 判断新游不是“开盘即巅峰” | 腾讯、辅助样本 |
| 销售费用率 | 销售费用 / 收入 | 判断增长是否依赖买量 | 三七、恺英、心动 |

## 3. 二级指标：用于解释竞争优势

### 3.1 长青资产质量

| 指标 | 说明 | 主要意义 |
| --- | --- | --- |
| 上线 5 年以上核心产品数 | 统计仍有重要流水贡献的产品 | 衡量长线运营底盘 |
| 上线 8 年以上核心产品数 | 更严格的长寿命标准 | 衡量真正的“老树开花”能力 |
| 单款长青产品重大版本频率 | 资料片、赛季、联动等更新频率 | 衡量运营深度 |
| 长青产品贡献毛利占比 | 长青产品对应毛利 / 游戏毛利 | 衡量现金牛质量 |

### 3.2 新游接续能力

| 指标 | 说明 | 主要意义 |
| --- | --- | --- |
| 已获版号重点产品数 | 已获批且具备上线预期的项目 | 反映短中期供给 |
| 重点新品 12-18 个月上线数 | 真实可能落地的产品数 | 比储备总数更有意义 |
| 重点新品品类匹配度 | 是否仍在公司优势赛道 | 判断成功概率 |
| 测试到上线兑现率 | 已测试产品最终上线比例 | 判断组织执行力 |
| 新品次年流水留存率 | 次年流水 / 首年流水 | 判断新游质量而非首发热度 |

### 3.3 商业化与流量质量

| 指标 | 说明 | 主要意义 |
| --- | --- | --- |
| 自然量占比 | 自然新增 / 总新增 | 判断品牌和社区优势 |
| 买量回收周期 | CAC 回收所需时间 | 判断增长质量 |
| 付费率 | 付费用户 / 活跃用户 | 衡量商业化深度 |
| ARPPU | 每付费用户平均收入 | 衡量高价值用户贡献 |
| 留存率 | 次日、7 日、30 日 | 判断内容吸引力与产品稳定性 |

### 3.4 研发工业化能力

| 指标 | 说明 | 主要意义 |
| --- | --- | --- |
| 研发人员占比 | 研发人数 / 总人数 | 体现研发导向 |
| 研发费用率 | 研发费用 / 收入 | 体现未来供给投入 |
| 人均研发产出 | 游戏收入或毛利 / 研发人数 | 衡量组织效率 |
| 核心工作室持续产出能力 | 重点团队是否连续推出可商业化产品 | 衡量组织可复制性 |
| 从立项到上线周期 | 平均研发周期 | 衡量项目管理效率 |

### 3.5 海外第二曲线

| 指标 | 说明 | 主要意义 |
| --- | --- | --- |
| 海外收入占比 | 海外收入 / 总收入 | 判断第二曲线是否成立 |
| 海外收入同比 | 海外收入增长速度 | 判断边际变化 |
| 海外核心产品数量 | 在海外有稳定流水的产品数 | 判断不是单项目驱动 |
| 海外自发行业务占比 | 自研自发行为主还是合作发行为主 | 判断能力深度 |
| 海外新品增量贡献 | 海外新品对收入增量贡献 | 判断未来弹性 |

## 4. 三级指标：用于校准估值是否便宜

| 指标 | 用法 |
| --- | --- |
| EV/EBIT | 衡量成熟利润资产估值 |
| EV/FCF | 衡量现金流资产估值 |
| PE | 辅助参考，但不能单独使用 |
| 净现金调整后 EV | 剥离账上现金后的经营资产估值 |
| 股东回报率 | 分红率 + 回购率，看总回报 |

## 5. 实战中的比较顺序

建议按下面顺序分析，而不是一上来横向拉表：

1. 先算网易自己的长青产品质量、现金流质量、资本回报。
2. 再用腾讯校准网易与平台型龙头之间应有的折价。
3. 再用 NEXON 校准长线运营型网游公司的估值中枢。
4. 最后才用国内中型游戏公司去辅助判断买量、留存、研发和版号这些单项指标。

## 6. 最值得跟踪的高频变化项

如果后续做持续跟踪，最值得每季度刷新的不是所有指标，而是这 8 项：

1. 游戏业务收入增速
2. 游戏业务利润率
3. 自由现金流
4. 分红与回购力度
5. 长青产品是否稳住
6. 新游上线节奏与兑现率
7. 销售费用率和买量依赖度
8. 海外收入占比与核心新品表现

## 7. 估值指标数据口径定义

下面这组定义不是教科书口径，而是为了让网易、腾讯、NEXON 和后续辅助样本尽可能按同一把尺子比较。

### 7.1 核心游戏营业收入

定义：

1. 自研游戏流水分成后的净收入。
2. 代理游戏与研发方分账后的净收入。
3. 游戏 IP 对外授权收入，包括保底和分成。
4. 与游戏强相关的电竞、赛事、周边等衍生收入。

推荐分子分母：

- 分子：上面四项合计后的净额。
- 分母：如果算占比，用总收入；如果算利润率，用同口径游戏收入。

为什么这样定义：

- 游戏公司真正进入报表的是净收入，不是玩家总流水。
- 只看总流水会高估平台分成和渠道分成后的真实经营规模。

常见误用：

- 用玩家总流水直接替代财报口径收入。
- 把云音乐、教育、电商等非游戏收入混进游戏主业。

### 7.2 游戏主业 EBIT

定义：

1. 核心游戏营业收入。
2. 减去游戏分部直接营业成本。
3. 再减去与游戏直接相关的研发、销售、管理费用。
4. 若公司不披露分部费用，则按更保守的分摊规则估算总部费用。

推荐分子分母：

- 分子：游戏主业 EBIT。
- 分母：核心游戏营业收入。

为什么这样定义：

- 对网易这类高现金流游戏公司，`EBIT` 比净利润更接近主业真实盈利能力。
- 直接用集团 `EBIT` 会被非游戏业务和非经营项目污染。

常见误用：

- 用集团营业利润直接替代游戏主业利润。
- 用按收入占比分摊费用的激进口径高估游戏分部利润率。

### 7.3 广义自由现金流

定义：

1. 经营现金流。
2. 减去 `PPE capex`。
3. 减去内容、版权、资本化研发或其他维持经营所必需的再投资。

推荐分子分母：

- 分子：广义 `FCF`。
- 分母：总收入，或核心游戏营业收入。

为什么这样定义：

- 对游戏公司，内容投入和版权投入本质上也是经营性再投资。
- 只扣硬件和服务器 `capex` 会高估可分配现金。

常见误用：

- 只用 `OCF - PPE capex` 作为唯一 `FCF`。
- 把一次性营运资本释放误认为长期现金流能力。

### 7.4 长青产品收入占比

定义：

1. 先定义长青产品篮子。
2. 推荐标准是：上线至少 5 年，或连续 3 个以上季度被管理层点名为核心成熟产品。
3. 统计这组产品的收入或流水贡献。

推荐分子分母：

- 分子：长青产品篮子收入。
- 分母：核心游戏营业收入。

为什么这样定义：

- 长青产品决定网易这类公司的现金流底盘和估值下限。
- 单看“有没有老产品”不如看“老产品贡献了多少收入”。

常见误用：

- 只数产品数量，不看收入贡献。
- 把短生命周期爆款也算进长青产品。

### 7.5 核心老游戏衰减速度

定义：

1. 选定核心老游戏篮子。
2. 用滚动 12 个月收入或流水，比较当前期与上年同期。
3. 重点看篮子整体，而不是单季度单产品波动。

推荐分子分母：

- 分子：当前滚动 12 个月老游戏收入。
- 分母：上一个滚动 12 个月老游戏收入。

为什么这样定义：

- 老游戏衰减不是看一个季度，而是看生命周期斜率。
- 对网易估值，老游戏衰减速度比某一季新游上线更决定中枢。

常见误用：

- 用单季度同比直接代表长期衰减率。
- 只看一款头部产品，忽略整个长青篮子。

### 7.6 新游兑现率

定义：

1. 统计某一时期上线的重点新品。
2. 看这些新品在上线 12 个月后，是否仍达到预设收入或流水门槛。
3. 达标数量占总样本数量的比例，即为兑现率。

推荐分子分母：

- 分子：上线 12 个月后仍有核心贡献的新品数量。
- 分母：样本期内上线的重点新品数量。

为什么这样定义：

- 真正重要的不是“发了多少新游”，而是“留下了多少新游”。
- 对网易来说，新游兑现率决定该不该给额外期权价值。

常见误用：

- 只看首月流水或首发排名。
- 把所有小体量项目和重点新品混在一起统计。

### 7.7 海外收入占比

定义：

1. 海外玩家产生、并由公司确认的净游戏收入。
2. 可以再拆成自研自发、联合发行、授权分成三类。

推荐分子分母：

- 分子：海外净游戏收入。
- 分母：核心游戏营业收入；必要时再看占总收入比例。

为什么这样定义：

- 对网易，海外第二曲线更应该看“游戏收入中的海外贡献”，而不是集团笼统地区口径。
- 这能更直接回答海外业务是否在重塑估值中枢。

常见误用：

- 把海外下载量或流水榜表现直接当成收入占比。
- 用集团地区收入口径代替游戏业务地区收入口径。

### 7.8 股东回报率

定义：

1. 现金分红。
2. 加回购金额。
3. 根据分析目的，可分别对 `FCF` 或市值求比。

推荐分子分母：

- 资本回报强度：`分红 + 回购 / 广义FCF`
- 股东回报率：`分红收益率 + 回购收益率`

为什么这样定义：

- 对网易，分红和回购是估值的重要组成部分，不是附属项。
- 同时看对 `FCF` 和对市值的比率，才能区分“有回报意愿”和“市场实际拿到多少”。

常见误用：

- 只看分红，不看回购。
- 用已授权回购额度替代实际回购金额。

### 7.9 净现金 / 市值

定义：

1. 现金及现金等价物。
2. 加短期投资和高流动性长期投资。
3. 减有息债务；如需更保守，可再扣除优先股和租赁负债。

推荐分子分母：

- 分子：净现金。
- 分母：当前市值。

为什么这样定义：

- 这个指标衡量的是网易估值底部的现金保护。
- 对高现金流、低杠杆公司尤其重要。

常见误用：

- 只看账面现金，不扣债务。
- 把经营性负债全部当成有息债务机械扣减。

### 7.10 EV / EBIT

定义：

1. 企业价值 `EV = 市值 + 有息债务 - 现金及高流动性投资`。
2. 分母优先用核心经营资产或游戏主业的 `EBIT`。

推荐分子分母：

- 分子：经营资产 `EV`。
- 分母：核心游戏 `EBIT`，或剔除非游戏业务后的经营 `EBIT`。

为什么这样定义：

- `EV/EBIT` 适合比较成熟盈利期游戏资产的经营价值。
- 能比 `PE` 更少受现金和非经营损益影响。

常见误用：

- 用集团 `EBIT` 去比游戏主业 `EV`。
- 不剔除净现金，直接拿市值去比利润。

### 7.11 EV / FCF

定义：

1. 分子仍为经营资产 `EV`。
2. 分母优先用统一口径的广义 `FCF`。

推荐分子分母：

- 分子：经营资产 `EV`。
- 分母：广义 `FCF`。

为什么这样定义：

- 对网易这种高现金流资产，`EV/FCF` 往往比 `PE` 更贴近股东真实回报。
- 如果长青产品护城河稳固，`EV/FCF` 是非常关键的估值锚。

常见误用：

- 现金流口径和同业不一致却硬比较倍数。
- 用阶段性高现金回款年份直接外推长期中枢。
