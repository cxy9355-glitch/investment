# 网易可比公司对比框架

更新时间：2026-03-28

## 1. 可比公司池调整原则

本次口径不再使用任天堂、Take-Two、Roblox 等海外主机或平台型公司作为核心可比。

原因很明确：

- 商业模式差异过大。
- 用户结构与渠道结构差异过大。
- 版号、买量、安卓渠道、国内 IP 生态等关键变量不同。
- 对网易未来 2-3 年经营判断的参考意义不如国内上市游戏公司。

但“国内上市游戏公司为主”不能机械理解为都和网易投资价值接近。

更准确的口径应是：

- 腾讯是国内主锚。
- NEXON 是国际网游运营锚。
- A 股/港股中型游戏公司主要用于辅助拆指标，而不是作为网易整体投资价值的主锚。

## 2. 建议可比公司池

### 主锚

#### 腾讯

- 角色定位：国内主锚。
- 主要用途：比较整体游戏业务质量、研发工业化、组织深度、全球化和资本配置。
- 关键提醒：腾讯的平台属性更强，因此不能简单拿估值倍数硬比，而要看网易与腾讯之间的结构性折价是否合理。

### 国际运营锚

#### NEXON

- 角色定位：国际网游运营锚。
- 主要用途：比较长线运营、现金流质量、海外结构和股东回报。
- 关键提醒：NEXON 比多数欧美主机或平台型公司更接近网易，因为其核心仍然是在线游戏长期运营。

### 辅助指标样本

#### 三七互娱

- 代表能力：买量发行、研运协同、海外发行、成熟手游工业化。
- 与网易的主要可比点：买量效率、发行体系、出海效率、股东回报。
- 主要差异：三七更偏买量驱动和发行能力，网易自研和长青产品属性更强。

#### 吉比特

- 代表能力：精品研发、利润质量、股东回报、项目选择克制。
- 与网易的主要可比点：高利润率、高现金流、研发导向。
- 主要差异：体量更小、产品集中度更高，平台和发行维度弱于网易。

#### 完美世界

- 代表能力：MMO 研发积累、端游与手游结合、全球化研运。
- 与网易的主要可比点：重度品类、研发体系、长周期产品。
- 主要差异：近年业绩波动更大，产品兑现稳定性弱于网易。

#### 恺英网络

- 代表能力：传奇类与 IP 类产品运营、买量商业化、利润释放。
- 与网易的主要可比点：产品商业化效率、股东回报、成熟产品运营。
- 主要差异：恺英在品类与用户结构上更偏下沉和 IP 改编，网易品牌与研发壁垒更强。

#### 心动公司

- 代表能力：TapTap 平台、社区流量、研发与分发联动。
- 与网易的主要可比点：研发能力、用户社区、产品口碑与自然量。
- 主要差异：心动的平台资产和社区属性更强，利润释放阶段性更弱。

### 观察样本

#### 游族网络

- 重点看全球化发行、SLG 赛道、老产品续航与新品储备兑现。

#### 中手游

- 重点看 IP 储备、授权改编、港股估值折价与流水兑现质量。

#### 创梦天地

- 重点看渠道、平台、IP 合作与线上娱乐生态，但经营质量波动较大，适合做辅助观察，不建议做核心估值锚。

## 3. 专业投资者更关心的对比框架

### 3.1 主分析层应该怎么用可比对象

如果目标是真正判断网易投资价值，建议按下面顺序使用可比对象：

1. 先用腾讯比较资产质量和组织上限。
2. 再用 NEXON 比较长线运营型网游公司的估值中枢和现金流质量。
3. 最后才用三七、吉比特、完美、恺英、心动、游族去拆单项指标。

### 3.2 财务质量对比

这是基础层，但不是终点。统一建议比较：

- 收入同比、3 年 CAGR。
- 毛利率。
- 营业利润率。
- 净利率。
- 经营现金流率。
- 自由现金流率。
- 研发费用率。
- 销售费用率。
- 净现金/市值。
- 分红率、回购强度、股东回报率。
- PE、EV/EBIT、EV/FCF。

### 3.3 产品竞争力对比

这是更重要的一层，但更适合拿来做“整体游戏业务质量”的支撑解释，而不是直接做拆块估值：

- 头部在营产品数量。
- 过去 3 年贡献主要流水的产品数量。
- 成熟产品组合稳定性。
- 增量产品上线节奏。
- 重点产品所在品类。
- 自研产品占比。
- 重点产品生命周期长度。
- 过去 3 年形成持续经营贡献的新增产品数。
- 管理层点名成熟产品驱动增长的频次。
- 核心产品组合对整体收入的支撑强度。

### 3.4 研发组织能力对比

建议重点看：

- 研发人员数量及占比。
- 研发费用率。
- 人均研发产出。
- 从立项到上线的平均周期。
- 项目储备梯队是否清晰。
- 是否存在稳定的核心制作人体系。
- 重点工作室的持续产出能力。

### 3.5 用户与商业化能力对比

建议重点看：

- 核心产品 DAU / MAU。
- 付费率。
- ARPPU。
- 留存率。
- 用户时长。
- 自然新增与买量新增占比。
- 销售费用率与投放效率。
- 是否依赖强买量换增长。
- 自然量与社区反哺能力。

### 3.6 内容供给与版号储备对比

建议重点看：

- 已获版号数量。
- 在研重点新品数量。
- 已测试待上线产品数量。
- 赛道覆盖广度。
- 核心 IP 自有化程度。
- 产品上线节奏是否连续。
- 从版号取得到正式上线的兑现率。

### 3.7 出海与第二增长曲线对比

建议重点看：

- 海外收入占比。
- 海外重点产品数。
- 海外收入区域分布。
- 海外自发行为主还是合作发行为主。
- 海外产品是否具备持续运营，而非一次性爆发。

## 4. 网易横向比较时最值得证明的命题

后续做研究时，重点不是证明“网易财务不错”，而是证明以下命题是否成立：

1. 网易的整体游戏业务质量，比国内多数游戏公司更稳、更强现金流。
2. 网易的研发和运营体系，不只是做出爆款，而是能持续支撑整体经营质量。
3. 网易对买量的依赖度低于典型发行驱动公司，因此利润质量更高。
4. 网易的海外能力和产品供给能力，足以支撑未来 2-3 年增长预期。
5. 网易账上净现金和股东回报政策，使其下行风险低于多数可比公司。
6. 与腾讯相比，网易虽然缺少超级平台，但其估值折价是否已经充分反映这一点。
7. 与 NEXON 相比，网易是否被低估了长线网游现金流的质量。
