## 1. 使用原则

这张表只服务于后续正式估值。

核心原则：

- `已填`：可以直接进入估值框架
- `代理值`：可以先做方向性判断，但不能在正式估值里伪装成精确值
- `待补`：当前资料不足，必须继续研究

## 3. 当前最关键的 5 个缺口

按对正式估值的影响程度排序：

1. 游戏主业营业利润率的更精确官方口径
2. 统一口径可比 FCF 的历史回看与同业对照
3. 海外增量对整体游戏业务的持续贡献
4. 游戏业务经营质量辅助信号的持续跟踪

## 4. 当前可直接支撑的估值动作

基于现有已填数据，已经可以做的事情：

- 用 `SOTP` 框架先把净现金单独加回
- 用 `EV / Revenue`、`EV / EBITDA`、`PE` 看市场对网易的相对折价
- 用 `可比 FCF` 做保守现金流估值区间
- 用 `现金及现金等价物 / 广义现金 / 净现金` 三层口径分开看资产负债表安全垫
- 先把网易放在“高现金流游戏主业 + 厚净现金 + 持续股东回报”框架里，而不是普通游戏股框架里

## 5. 当前仍不能直接下结论的部分

现阶段还不能严谨下结论的点：

- 网易游戏主业应拿多高的 `EV / EBIT` 主倍数
- 海外第二曲线能否稳定支撑整体游戏业务增长中枢
- 股东回报是否足以支撑更高的估值底部

这些都依赖于上面 5 个核心缺口继续补数。
