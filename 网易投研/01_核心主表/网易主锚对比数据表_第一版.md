# 网易主锚对比数据表（第一版）

更新时间：2026-03-29

## 1. 口径说明

- 这张表优先使用截至 `2026-03-28` 已披露的 `FY2025 全年业绩公告 / Annual Results / Earnings Release`。
- `正式年报` 与 `全年业绩公告` 仍然分开看，不能混用。
- 本表第一版先填“已直接披露或可低争议推导”的核心数据。
- 本表当前分成三层：
  - `A. 集团层`：看资本市场定价、股东回报、现金流和资本配置。
  - `B. 游戏业务层`：网易、腾讯、NEXON 必须尽量按游戏业务口径对齐比较。
  - `C. 辅助样本层`：只用来拆解经营画像和单项指标，不作为网易整体估值主锚。
- 因此：
  - 网易、腾讯与 NEXON 在 `A. 集团层` 可以比较“上市主体”口径。
  - 真正判断网易是否被低估，核心仍要回到 `B. 游戏业务层`。
- 币种暂不统一折算：
  - 网易、腾讯：人民币
  - NEXON：日元
- 市值、EV、净现金/市值等估值层指标暂不在本表中硬填，后续需要配合同一时点股价口径单独补。
- 港股公司优先引用中文披露件：
  - 网易：`网易_2025Q4及全年业绩公告_官方中文版.pdf`
  - 腾讯：`腾讯_2025全年业绩公告_官方中文版.pdf`
- 辅助样本层当前优先复用本地 `原始资料\可比公司` 中已归档的正式报告；由于这些本地样本当前以 `FY2024` 为主，所以辅助样本层先用 `FY2024` 观察经营画像，不与主锚 `FY2025` 机械混成同一估值口径。

## 9. 当前最关键的投资判断

| 判断项 | 网易结论 | 备注 |
| --- | --- | --- |
| 游戏业务整体质量是否被低估 | 大概率是 | 网易游戏收入占比高、利润率高、净现金厚，但市场常把其当“缺平台的中国游戏股”看待 |
| 游戏业务增长支撑是否被低估 | 有争议但并非弱 | 2025 年新品与海外增量已提供支持，但更关键的是它们能否稳定进入整体收入中枢 |
| 现金流质量是否被低估 | 是 | 经营现金流、净现金、分红回购能力都很强 |
| 平台折价是否已充分反映 | 需要用腾讯进一步校准 | 腾讯应有溢价，但网易当前折价是否过大，是核心问题 |
| 与 NEXON 相比是否低估了长线网游现金流 | 值得重点验证 | NEXON 的估值中枢更适合校准“高现金流长线网游运营商”该拿什么估值 |
| 当前估值的主要矛盾 | 整体游戏业务质量很强，但市场对增长持续性和平台折价更敏感 | 这也是后续最该补数据证明的部分 |

## 10. 当前仍需优先跟踪的 8 个事项

1. 网易游戏分部营业利润率的更精确官方口径，而不只是近似 EBIT 率。
2. 网易海外增量对整体游戏业务收入中枢的持续贡献。
3. 网易买量依赖度与销售费用率趋势。
4. 网易经营质量代理信号向收入和利润指标的转化验证。
5. 腾讯、网易、NEXON 三者在“整体游戏业务质量 + 股东回报”上的统一估值锚。
6. 腾讯 FY2025 现金分红总额在 `record date` 锁定股本后的最终确认。
7. 辅助样本层从 `FY2024` 向 `FY2025` 的正式披露更新。
8. 非游戏业务同一时点、同一币种、同一权益口径的估值补充。
