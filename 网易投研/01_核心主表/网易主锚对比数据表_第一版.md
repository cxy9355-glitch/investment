初步观察：

- 在当前市场口径下，网易的 `Trailing PE`、`Forward PE`、`EV / Revenue` 和 `EV / EBITDA` 均明显低于腾讯。
- 网易与 NEXON 相比，当前 `PE` 也显著更低，而 `EV / EBITDA` 更接近。
- 这意味着市场可能已经给了网易明显的平台折价，但是否还额外给了过度的“增长 / 新游折价”，需要继续用主业质量数据验证。

## 9. 当前最关键的投资判断

| 判断项 | 网易结论 | 备注 |
| --- | --- | --- |
| 长青资产质量是否被低估 | 大概率是 | 网易游戏收入占比高、利润率高、净现金厚，但市场常把其当“缺平台的中国游戏股”看待 |
| 新游接续能力是否被低估 | 有争议但并非弱 | 2025 年 Where Winds Meet、Marvel Rivals 提供了增量，但仍需证明可持续而非阶段性；这是和腾讯、NEXON 拉开估值差异的核心点 |
| 现金流质量是否被低估 | 是 | 经营现金流、净现金、分红回购能力都很强 |
| 平台折价是否已充分反映 | 需要用腾讯进一步校准 | 腾讯应有溢价，但网易当前折价是否过大，是核心问题 |
| 与 NEXON 相比是否低估了长青网游现金流 | 值得重点验证 | NEXON 的估值中枢更适合校准“高现金流长线网游运营商”该拿什么估值 |
| 当前估值的主要矛盾 | 长青现金流很强，但市场对新游持续性和平台折价更敏感 | 这也是后续最该补数据证明的部分 |
