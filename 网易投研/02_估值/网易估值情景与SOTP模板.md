# 网易估值情景与 SOTP 模板

更新时间：2026-03-28

## 1. 使用说明

这份模板不是最终估值结果，而是后续正式估值时要直接填数的工作底稿。

推荐顺序：

1. 先填主业输入变量
2. 再填主业估值区间
3. 再填非游戏业务
4. 最后加净现金
5. 再做三情景汇总

口径约束：

- 本模板各指标的统一定义，默认以 `网易估值备忘录_框架.md` 第 `4` 节“估值指标数据口径定义”为准。
- 如果后续取数只能拿到代理值，必须在备注中显式标注，不能和正式口径混填。

## 8. 估值结论模板

## 9. 当前默认估值逻辑

- 游戏主业是核心
- 非游戏业务保守处理
- 净现金单独加回

- NEXON 更适合校准长线网游运营资产中枢
- 国内中型游戏公司仅用于辅助说明单项指标，不作为网易整体估值主锚

第一版结论的核心含义是：

- 当前市场至少没有充分反映网易高现金流游戏主业、厚净现金和持续股东回报的组合价值。
- 但市场仍在持续给网易平台折价，并对新游与海外的可持续性保持谨慎。
- 所以后续最值得继续补的，不是把游戏业务拆得更细，而是继续验证整体游戏业务增长中枢、利润率中枢和海外增量能否持续进入报表。

## 11. 第一版估值计算过程

这一节只展开“怎么算”，方便后续复核，不新增新的结论。

### 11.1 基准情景计算过程

步骤 1：先假设未来 12 个月游戏收入

- `FY2025` 游戏收入：`RMB 92.1bn`
- 基准假设增速：`+6%`
- 所以：
  - `未来 12 个月游戏收入 = 92.1 * 1.06 = RMB 97.7bn`

步骤 2：计算游戏主业 `EBIT`

- 基准 `EBIT` 利润率：`34.25%`，对应模板里的 `34.0% - 34.5%` 中枢
- 所以：
  - `游戏主业 EBIT = 97.7 * 34.25% = RMB 33.5bn`

步骤 3：用 `EV/EBIT` 算主业价值

- 基准主业倍数：`14x`
- 所以：
  - `游戏主业 EV = 33.5 * 14 = RMB 469bn`

步骤 4：用 `FCF` 做交叉验证

- 基准游戏主业 `FCF` 利润率：`39%`
- 所以：
  - `游戏主业 FCF = 97.7 * 39% = RMB 38.1bn`
- 基准 `EV/FCF` 倍数：`10.5x`
- 所以：
  - `EV/FCF 对应 EV = 38.1 * 10.5 = RMB 400bn`

步骤 5：给非游戏业务保守估值

- 有道：`RMB 2.3bn`
- 云音乐：`RMB 26.0bn`
- 创新及其他：`RMB 0bn`
- 所以：
  - `非游戏业务合计 = RMB 28.3bn`

步骤 6：加回净现金

- `FY2025` 已披露净现金：`RMB 163.5bn`

步骤 7：汇总股权价值

- `股权价值 = 游戏主业 EV + 非游戏业务估值 + 净现金`
- `= 469 + 28.3 + 163.5`
- `= RMB 660.8bn`

步骤 8：换算每股价值

- 当前市值：`HKD 557.8bn`
- 当前股价：`HKD 173.1`
- 反推普通股股本：`557.8 / 173.1 = 约 32.23 亿股`
- 汇率近似假设：`HKD/RMB = 0.92`
- 所以：
  - `股权价值（HKD） = 660.8 / 0.92 = HKD 718.3bn`
  - `每股价值 = 718.3 / 32.23 = 约 HKD 222.9`

### 11.2 乐观情景计算过程

- 游戏收入：`92.1 * 1.12 = RMB 103.2bn`
- 游戏主业 `EBIT`：`103.2 * 35.5% = RMB 36.6bn`
- 主业 `EV`：`36.6 * 16 = RMB 586bn`
- 非游戏业务：`RMB 33.0bn`
- 净现金：`RMB 163.5bn`
- 股权价值：`586 + 33.0 + 163.5 = RMB 782.5bn`
- 对应每股价值：`782.5 / 0.92 / 32.23 = 约 HKD 263.9`

### 11.3 保守情景计算过程

- 游戏收入：`RMB 92.1bn`
- 游戏主业 `EBIT`：`92.1 * 33.5% = RMB 30.9bn`
- 主业 `EV`：`30.9 * 12 = RMB 370bn`
- 非游戏业务：`RMB 23.0bn`
- 净现金：`RMB 163.5bn`
- 股权价值：`370 + 23.0 + 163.5 = RMB 556.5bn`
- 对应每股价值：`556.5 / 0.92 / 32.23 = 约 HKD 187.7`

### 11.4 为什么第一版主结论先采用 EV/EBIT

- `EV/FCF` 交叉验证得到的主业价值普遍低于 `EV/EBIT`
- 这反映市场仍在给网易平台折价和增长折价
- 第一版若直接用 `EV/FCF` 做主结论，会过度放大保守性
- 所以当前更合理的处理是：
  - `EV/EBIT` 作为主锚
  - `EV/FCF` 作为保守校验
  - `DCF` 作为 sanity check

## 12. 当前仍需继续补强的口径

第一版估值已经能跑，但还有几项关键口径值得继续补强，用来提高整体游戏业务估值的稳健性。

### 12.1 游戏主业营业利润率的正式口径

- 当前状态：已有 `33.5% - 35.0%` 的保守代理区间
- 为什么还缺：官方不披露分部营业费用和分部 `EBIT`
- 怎么补：
  - 持续跟踪后续年报附注
  - 如果未来披露更细分部费用，再用正式口径替换当前代理值

### 12.2 游戏主业 FCF 的统一口径

- 当前状态：已有广义 `FCF` 主口径，但仍属于研究口径
- 为什么还缺：内容投入、版权投入与经营性再投资的归类仍需持续校验
- 怎么补：
  - 按同一逻辑回看历史期
  - 与腾讯、NEXON 的现金流口径做统一对照

### 12.3 海外增量对整体游戏业务的持续贡献

- 当前状态：能确认海外第二曲线增强，但还不能精确量化为单一占比
- 为什么还缺：官方没有披露 FY2025 海外游戏收入占比
- 怎么补：
  - 继续跟踪核心出海产品的收入与活跃代理
  - 更关注其是否持续进入整体游戏业务收入中枢，而不是追求单点精确占比

### 12.4 非游戏业务估值精度

- 当前状态：第一版只做了保守近似
- 为什么还缺：有道、云音乐、创新及其他业务还没有统一成同一时点、同一货币的精细口径
- 怎么补：
  - 单独建一页“非游戏 SOTP 补充表”
  - 统一市场时点、币种和折价逻辑
