# 贵州茅台（600519.SH）DCF 估值草稿

> 估值日期：2026-03-22  
> 口径说明：使用 **2024 年年报** 已披露的审计数据作为基期，采用 **股东自由现金流（FCFE 近似）** 做简化 DCF。由于贵州茅台有息负债极少、账上现金充裕，这个简化口径通常比严格 FCFF 更直观。  
> 单位：除每股价值外，以下金额均为 **人民币亿元**。

---

## 1. 先说结论

在本文的**基准情景**下：

- 2024A 基期自由现金流：**877.85 亿元**
- 未来 5 年自由现金流增速假设：**9% / 8% / 7% / 6% / 5%**
- 折现率（r）：**8.0%**
- 永续增长率（g）：**3.0%**
- 得到股权价值约：**2,160.32 亿元**
- 总股本约：**12.561978 亿股**
- 对应每股内在价值约：**1,719.73 元/股**

这不是一个“精确答案”，而是一个**可复核、可改假设、可继续迭代**的 DCF 框架。

---

## 2. 原始数据：从 2024 年报取数

本文直接使用贵州茅台 2024 年年度报告中的以下核心数据：

### 2.1 经营层面

- 营业总收入：**1,708.99 亿元**
- 归母净利润：**862.28 亿元**
- 经营活动现金流净额：**924.64 亿元**

### 2.2 资本开支

年报现金流量表中“购建固定资产、无形资产和其他长期资产支付的现金”为：

- 资本开支（Capex）：**46.79 亿元**

### 2.3 股本

- 股本：**12.561978 亿股**

### 2.4 负债情况

年报显示，公司几乎没有传统意义上的有息借款：

- 短期借款：未列示
- 长期借款：未列示
- 债券应付：未列示
- 租赁负债存在，但规模很小

因此，这里采用 **FCFE 近似法**，直接用“经营现金流 - 资本开支”作为对股东可分配现金流的近似值，不再复杂展开净债务调整。

---

## 3. 估值方法：为什么这里用 FCFE 近似

严格 DCF 常见有两种做法：

1. **FCFF（企业自由现金流）**：折现到企业价值，再加减净债务得到股权价值；
2. **FCFE（股东自由现金流）**：直接折现到股权价值。

考虑到贵州茅台：

- 现金创造能力极强；
- 有息负债极少；
- 资本开支相对经营现金流占比不高；
- 业务模式成熟，现金流质量高；

所以这里用下面这个简化公式：

```text
FCFE ≈ 经营活动现金流净额 - 资本开支
```

即：

```text
2024A 自由现金流 ≈ 924.64 - 46.79 = 877.85 亿元
```

---

## 4. 关键假设

DCF 最重要的不是公式，而是**假设是否保守、是否自洽**。

### 4.1 收入/自由现金流增长假设

我给贵州茅台设定一个“高质量龙头，但增速逐步回落”的路径：

- 2025E：**9%**
- 2026E：**8%**
- 2027E：**7%**
- 2028E：**6%**
- 2029E：**5%**

这个假设背后的逻辑：

- 茅台品牌力极强，提价能力、产品结构升级能力都优于大多数消费股；
- 但公司体量已经很大，不宜长期假设双位数高增；
- 因此前高后低、逐步收敛，更符合成熟消费龙头的估值框架。

### 4.2 折现率（r）

基准情景用 **8.0%**。

理由：

- 茅台属于经营稳定、盈利能力强、资产负债表优秀的核心消费龙头；
- 但 A 股权益资产仍需风险补偿；
- 用 8% 作为股权现金流折现率，属于偏审慎但不算苛刻的设定。

### 4.3 永续增长率（g）

基准情景用 **3.0%**。

理由：

- 永续增长不能高于长期名义经济增速太多；
- 对成熟白酒龙头而言，3% 是较常见、相对中性的长期假设。

## 5. 具体计算过程

## 5.1 基期自由现金流

```text
FCF_2024 = CFO_2024 - Capex_2024
         = 924.64 - 46.79
         = 877.85 亿元
```

## 5.2 未来五年自由现金流预测

按上述增速推演：

| 年度 | 增速 | 预测 FCF（亿元） | 折现因子（8%） | 折现后价值（亿元） |
|---|---:|---:|---:|---:|
| 2025E | 9% | 956.86 | 1 / 1.08^1 | 885.98 |
| 2026E | 8% | 1,033.40 | 1 / 1.08^2 | 885.98 |
| 2027E | 7% | 1,105.74 | 1 / 1.08^3 | 877.77 |
| 2028E | 6% | 1,172.09 | 1 / 1.08^4 | 861.52 |
| 2029E | 5% | 1,230.69 | 1 / 1.08^5 | 837.59 |

未来 5 年现金流现值合计：

```text
PV(2025-2029) = 885.98 + 885.98 + 877.77 + 861.52 + 837.59
              = 4,348.84 亿元
```

## 5.3 永续价值（Terminal Value）

采用 Gordon Growth Model：

```text
TV_2029 = FCF_2030 / (r - g)
        = FCF_2029 × (1 + g) / (r - g)
        = 1,230.69 × 1.03 / (8% - 3%)
        = 25,352.26 亿元
```

再折现回今天：

```text
PV(TV) = 25,352.26 / 1.08^5
       = 17,254.32 亿元
```

## 5.4 股权价值

```text
Equity Value = PV(2025-2029) + PV(TV)
             = 4,348.84 + 17,254.32
             = 21,603.16 亿元
```

## 5.5 每股内在价值

```text
每股价值 = 股权价值 / 总股本
        = 21,603.16 / 12.561978
        ≈ 1,719.73 元/股
```

## 6. 敏感性分析

DCF 对折现率和永续增长率非常敏感，所以必须看区间，不要只看单点。

### 每股内在价值敏感性表（元/股）

| 折现率 \ 永续增长率 | 2% | 3% | 4% |
|---|---:|---:|---:|
| 7% | 1,780.89 | 2,154.59 | 2,777.43 |
| 8% | 1,479.69 | 1,719.73 | 2,079.78 |
| 9% | 1,264.68 | 1,429.93 | 1,661.27 |

解读：

- 如果你更保守，把折现率提到 **9%**、永续增长率降到 **2%~3%**，估值会回落到 **1,265~1,430 元/股**；
- 如果你认为茅台长期竞争优势更强，接受 **7%** 折现率与 **3%** 永续增长，估值可以到 **2,155 元/股** 左右；
- 所以更合理的理解方式不是“它值 1719.73”，而是：**在一组常见假设下，内在价值大致落在 1,400~1,800 元/股附近，乐观情景更高，保守情景更低。**

## 7. 这个 DCF 的优点与局限

### 7.1 优点

- 直接基于年报现金流，口径清晰；
- 茅台现金流质量高，适合用 DCF 做中长期锚定；
- 简化模型容易复核，也容易替换假设。

### 7.2 局限

- 这里用的是 **FCFE 近似**，没有严格拆分营运资本、税后经营利润、非经营性资产等细项；
- 没有单独处理价格带升级、直营占比变化、系列酒增长、分红率变化等经营变量；
- DCF 对终值高度敏感，因此它更适合作为“估值区间工具”，而不是绝对真理；
- 2025 年及之后的预测本质上是主观判断，不是公司给出的业绩承诺。

## 8. 如果你要继续精细化，我建议下一步这么做

如果你想把这版继续升级成“能拿去投委会讨论”的版本，可以继续加三层：

1. **收入拆分**：
   - 茅台酒销量
   - 吨价/提价
   - 直销 vs 批发占比
   - 系列酒增速

2. **利润率拆分**：
   - 毛利率
   - 销售费用率
   - 税金及附加
   - 净利率稳定区间

3. **现金流拆分**：
   - 营运资本变动
   - 资本开支常态水平
   - 分红率与留存现金

这样可以把“拍脑袋增长率”改造成“经营驱动 DCF”。

## 9. 本文使用的核心公式汇总

```text
(1) 自由现金流近似：
FCFE ≈ CFO - Capex

(2) 显性期折现：
PV_t = FCF_t / (1 + r)^t

(3) 永续价值：
TV_n = FCF_(n+1) / (r - g)

(4) 总股权价值：
Equity Value = Σ PV_t + PV(TV)

(5) 每股价值：
Value per Share = Equity Value / Shares Outstanding
```
