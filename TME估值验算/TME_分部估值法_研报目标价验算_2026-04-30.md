# TME 分部估值法研报目标价验算

日期：2026-04-30

## 1. 采用数据

| 项目 | 数值 |
|---|---:|
| 当前价 | 9.43 美元/ADS |

| 正文目标价 | 11.6 美元/ADS |

| 2026E 经调整净利润 | 10,811 百万元人民币 |

| 项目 | 2025A |
|---|---:|

## 2. 目标价反推

分部估值法公式：

```text
目标股权价值 = 在线音乐净利润 * 20 + 社交娱乐净利润 * 5
总净利润 = 在线音乐净利润 + 社交娱乐净利润
```

设在线音乐贡献 2026E 净利润比例为 `p`，则目标隐含 PE 为：

```text
隐含 PE = 20p + 5(1-p) = 5 + 15p
```

用研报截图 `2026E 经调整净利润 108.11 亿元人民币`、`15.48 亿 ADS`、TME 官方 2025 折算汇率 `6.9931` 测算：

| 目标价 | 隐含整体 PE | 反推在线音乐净利润占比 |
|---:|---:|---:|
| 11.6 美元 | 11.62x | 44.1% |
| 12.10 美元 | 12.12x | 47.4% |

也就是说，研报若要用 `20x + 5x` 得到 11.6-12.1 美元，必须隐含：

```text
在线音乐只贡献 2026E 净利润的约 44%-47%
社交娱乐贡献 2026E 净利润的约 53%-56%
```

## 3. 这个隐含假设是否合理

按官方 2025 实际收入：

```text
在线音乐收入占比 = 26.73 / 32.90 = 81.2%
社交娱乐及其他收入占比 = 18.8%
```

如果假设 2026 年社交娱乐继续按 2025 年 -7.3% 下降，则：

```text
2026E 社交娱乐收入 = 6.18 * (1 - 7.3%) = 5.73 十亿元
2026E 在线音乐收入 = 36.07 - 5.73 = 30.34 十亿元
2026E 在线音乐收入占比 = 84.1%
```

在这个收入结构下，若目标价为 11.6 美元，在线音乐只贡献 44.1% 利润，则意味着：

```text
在线音乐净利率 / 社交娱乐净利率 ≈ 0.15x
```

若目标价为 12.10 美元，则：

```text
在线音乐净利率 / 社交娱乐净利率 ≈ 0.17x
```

这个假设非常反直觉：在线音乐收入已经占 80% 以上，且公司官方解释毛利率提升来自音乐订阅、广告增长，以及社交娱乐分成比例下降；如果在线音乐只贡献不到一半利润，等于假设社交娱乐的净利率是在线音乐的 6 倍左右。

## 4. 情景复算

用同一组 `20x + 5x` PE 和 `2026E 经调整净利润 108.11 亿元人民币`，只改变在线音乐利润占比：

| 在线音乐净利润占比 | 隐含整体 PE | 目标价 |
|---:|---:|---:|
| 84.1%（按收入占比近似） | 17.62x | 17.59 美元 |
| 75% | 16.25x | 16.23 美元 |
| 65% | 14.75x | 14.73 美元 |
| 55% | 13.25x | 13.23 美元 |
| 50% | 12.50x | 12.48 美元 |
| 45% | 11.75x | 11.73 美元 |

## 5. 结论

研报这页的分部估值法不是单纯四则运算写错，而是**关键利润拆分缺失，且目标价隐含的利润拆分很难自洽**。

如果坚持 `在线音乐 20x + 社交娱乐 5x`：

1. `11.6 美元` 目标价只能在“在线音乐利润占比约 44%”时成立。
2. `12.10 美元` 目标价只能在“在线音乐利润占比约 47%”时成立。
3. 但在线音乐 2025 收入占比已达 81.2%，按 2026E 总收入推演可能接近 84%。
4. 因此，除非研报另有未披露的分部利润假设，否则目标价偏低，或者分部利润分配错了。

我认为更稳妥的判断是：

```text
这页分部估值链条不完整；
用官方收入结构校验，11.6/12.1 美元目标价对应的在线音乐利润占比过低；
若按较正常的在线音乐利润占比 65%-75% 测算，目标价应约为 14.7-16.2 美元；
若按收入占比近似，目标价约为 17.6 美元。
```

## 6. 已确认的其他口径问题

1. 左侧目标价 `12.10 美元` 与正文末尾 `11.6 美元` 不一致。
2. 左侧 `12.10 / 9.43 - 1 = 28.3%`，所以 28.3% 上行空间只对应 12.10，不对应 11.6。
3. 表中 2026E PE `10.5x` 反推汇率约为 7.85，和 TME 官方 2025 折算汇率 6.9931 不一致，也高于常用美元兑人民币口径。
