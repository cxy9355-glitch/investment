# 汽车行业研究目录说明

本目录当前先收口到 `2026Q1` 的整车公司季报原文。

## 当前内容

- `2026Q1季报原文/整车公司_Sina原文/`
  - 每家公司一份 `pdf`
  - 同步保存对应的 `html` 原文页
  - 以及便于快速全文检索的 `txt`
- `2026Q1季报原文/整车公司_report_index.csv`
  - 整车公司清单、公告标题、日期、详情页链接、本地文件路径

## 说明

这批文件优先解决“本地可直接打开看原文”。

在当前机器网络环境下，`CNINFO / SSE / SZSE / HKEXnews` 的静态 PDF 域名会被远端直接断开，无法稳定直连下载；因此本批落地改走新浪财经公告原文页中可访问的 PDF 链接与全文页镜像，保证研究目录先可用。
