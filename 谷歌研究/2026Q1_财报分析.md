### 净利润增长的拆解

净利润从 345.40 亿 → 625.78 亿，增加 **280.38 亿**：

| 来源 | 贡献（亿） | 说明 |
|---|---|---|
| 营业利润增长 | +90.90 | 核心业务赚钱能力提升 |
| 股权投资净收益暴增 | +271.57 | 非市场化股权（SpaceX/Waymo等）公允价值飙涨 |
| 利息、外汇、其他调整 | -6.24 | 利息支出增加（发债310亿） |
| 所得税增加 | -75.85 | 利润大涨导致税负翻倍 |

**但真相是**：Alphabet 自己披露，股权收益中 **$287 亿是税后净影响**（贡献 EPS $2.35）。

剔除股权投资后：
- **调整后净利润 ≈ $625.78 - $287.15 = ~$338.6 亿**
- 调整后 EPS ≈ $2.62（vs 分析师预期 $2.63，**实际微幅 miss**）

**结论**：81% 的利润暴增主要是股权投资收益驱动，经营层面利润增长 30%，两者都需要看清楚。

### 2.3 分部解析

#### Google Services (+79 亿营业利润增量，占 87%)

**搜索 (+97 亿营收增量)** 是绝对主力：
- AI Overviews 和 AI Mode 驱动查询量创历史新高
- 19% 增速在万亿美元营收体量（年度化 ~$2,400 亿）上实现，极其惊人
- 广告主对 AI 搜索改版的接受度没有问题

**订阅/平台/设备 (+20 亿营收)**：
- 付费订阅总数达到 3.5 亿（YouTube + Google One 驱动）
- YouTube 订阅增速已超过 YouTube 广告增速
- Pixel 设备贡献有限但稳定

**YouTube 广告 (+9.6 亿)**：
- 11% 增速相对稳健，但**miss 了分析师预期**（预期约 $100 亿）
- Shorts 货币化仍在爬坡阶段

**Google Network (-2.9 亿)**：
- 持续结构性萎缩（第三方广告网络），非核心

#### Google Cloud (+44 亿营业利润增量，占 49%)

**这是本季度的王炸**：

| 指标 | 2025Q1 | 2026Q1 | 
|---|---|---|
| 营收 | 122.60 | 200.28 |
| 增速 | — | **+63.3%** |
| 营业利润 | 21.77 | 65.98 |
| 营业利润率 | 17.8% | **32.9%** |

- 营收从 122 亿 → 200 亿，单季首次突破 **$200 亿**
- 利润率从 17.8% → 32.9%，**几乎翻倍**（规模效应+AI 产品高毛利）
- 企业 AI 解决方案首次成为 Cloud 的**第一大增长引擎**，超过传统 GCP
- Gemini Enterprise 付费月活用户环比增长 **40%**
- Cloud 积压订单（backlog）**接近翻倍**，达到 **$4,600 亿**
- API 处理量达到每分钟 160 亿 tokens，环比增长 60%

Google Cloud 的利润贡献（+44 亿）已经超过了营业利润总增量（+91 亿）的 **48%**。考虑到它只占总营收的 18%，利润杠杆效应极其突出。

#### Other Bets 和 Alphabet-level（吃掉了 30% 的经营利润增量）

| 项目 | 增量（亿） | 说明 |
|---|---|---|
| Other Bets 亏损扩大 | -8.74 | Waymo（500K 次/周无人驾驶）烧钱 |
| Alphabet-level 扩大 | -23.64 | AI 研发共享中心（Gemini 等基础模型训练） |

这两项合计吃掉 **32.4 亿**，抵消了 Service + Cloud 增量的 26%。Pichai 明确表示"我们仍在算力紧张状态，如果产能跟得上收入还会更高"——所以 AI 研发投入短期内不会缩减。

## 三、利润为什么暴增？完整归因

净利润从 345 → 626 亿美元（+81%），完整归因如下：

### 第一层：经营层面（可持续的部分）

```
营业利润: 306 → 397 亿 (+91亿, +30%)
├── Google Services: +79亿  ← 搜索AI化+订阅增长
├── Google Cloud:    +44亿  ← 63%收入增长+利润率翻倍  
├── Other Bets:       -9亿  ← Waymo烧钱
└── Alphabet-level:  -24亿  ← AI基础模型研发
```

经营利润增长 30%，**这是真实、可持续的部分**。

### 第二层：非经营层面（一次性/波动性）

```
其他收入: 112 → 377 亿 (+265亿, +237%)
├── 股权投资净收益: +271亿  ← 非上市股权（SpaceX等）公允价值重估
├── 利息收入:        +4亿
├── 利息支出:        -5亿   ← 发债310亿的代价
└── 其他:            -5亿
```

这 $271 亿是非现金、非经营收益。来自非市场化证券的公允价值暴涨（可能是 SpaceX、Waymo 的估值上修等）。**不可持续、不可预测，但代表了 Alphabet 投资版图的实际价值增长**。

### 第三层：税盾效应

```
所得税: 72 → 148 亿 (+76亿, +105%)
有效税率: 17.3% → 19.2%
```

税率略有上升，但绝对值暴增是因为利润基数翻倍。

## 四、值得注意的风险信号

### 1. 资本开支大幅飙升
- Q1 CapEx: **$357 亿**（去年同期 $172 亿，+107%）
- 全年指引上调至 **$1,800-1,900 亿**（之前 $1,750-1,850 亿）
- 2027 年 CapEx 预计"显著增长"
- 自由现金流被压缩：Q1 FCF 仅 **$101 亿**（去年同期 $189 亿）

### 2. 算力瓶颈
Pichai 明说 "我们算力受限，产能跟上的话 Cloud 收入会更高"。这意味着收入增长受到物理供给限制，不是需求问题。

### 3. YouTube 增长放缓
YouTube 广告 11% 增速 miss 预期，Shorts 货币化仍在爬坡。

### 4. 员工数反弹
19.5 万人，同比增加约 9,000 人（+4.8%），裁员红利消退。

### 5. 资产负债扩张
总资产从 5,953 亿 → 7,039 亿，长期债务从 465 亿 → 775 亿（发了 310 亿债）。净现金（现金+证券-负债）实际上在恶化。

## 六、总结

**利润暴增 81% 的真相**：

| 分项 | 如果只看 81% 你会以为… | 实际情况 |
|---|---|---|
| 经营利润 | — | +30%，也很优秀但没那么夸张 |
| 股权投资 | 赚了 | 非现金公允价值变动 $287 亿 |
| 核心业务增速 | — | 营收 +22%，搜索 +19%，Cloud +63% |
| Cloud 利润质量 | — | 利润率从 18% → 33%，真正的经营杠杆 |
| CapEx | — | 翻倍到 $357 亿/季，FCF 被严重压缩 |
| 剔除股权收益后 EPS | — | $2.62，实际上微幅 miss 了分析师预期 |

**一句话**：Google 的真实经营利润增长是 30%（而非报表的 81%）。Cloud 从"烧钱基础设施"变成了"利润引擎"。搜索 AI 化进展顺利（19% 增速 + 历史最高查询量）。但 CapEx 翻倍、FCF 被压缩、YouTube 广告放缓是不可忽视的阴影。整体看是 **A+ 级别的经营季报 + 意外的股权收益红利**。
