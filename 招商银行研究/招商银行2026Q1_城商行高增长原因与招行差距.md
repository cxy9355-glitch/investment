## 2. 先给结论

这几个头部城商行 2026Q1 增长好看，核心不是因为它们突然有了比招行更好的商业模式，而是三件事叠加：

1. **资产负债表扩张更猛。**  
   宁波贷款 +9.45%、存款 +10.89%；苏州贷款 +10.83%、存款 +12.96%；杭州贷款 +7.57%、存款 +5.50%。招行贷款只 +2.84%、存款 +1.25%。

2. **增长主要压在对公和区域信贷，而不是零售。**  
   宁波公司贷款 +16.58%，个人贷款只 +0.18%；苏州公司贷款 +13.36%，个人贷款 +0.56%；杭州公司金融条线贷款不含贴现 +14.00%，零售金融条线贷款增长很小。招行也是公司贷款 +6.98%、零售贷款 -1.00%，但招行体量太大，对公增长无法完全抵消零售熄火。

3. **部分银行确实在更积极消耗资本。**  
   苏州 RWA 较年初 +10.37%，CET1 从 9.53% 降到 8.97%；宁波 RWA 较年初 +5.59%，CET1 从 9.34% 降到 9.25%。招行 CET1 仍高达 14.13%，贷款增长偏温和，说明不是资本不够，而是没有把资本像城商行那样快速打出去。

所以回答你的问题：**招行增长慢，部分是资本扩张不激进，但根因不是“资本约束”，而是“优质高收益资产找不到足够多，尤其零售资产没有恢复”。**

## 3. 城商行为何增长好看

### 3.1 宁波银行：对公贷款高增 + 手续费爆发

| 项目 | 2026Q1/期末 | 变化 |
|---|---:|---:|
| 营收 | 203.84亿 | +10.21% |
| 归母净利润 | 81.81亿 | +10.30% |
| 利息净收入 | 146.88亿 | +14.44% |
| 手续费及佣金净收入 | 25.75亿 | +81.72% |
| 贷款及垫款 | 18,970.45亿 | +9.45% |
| 公司贷款 | 12,510.63亿 | +16.58% |
| 个人贷款 | 5,354.61亿 | +0.18% |
| 客户存款 | 22,453.06亿 | +10.89% |
| 净息差 | 1.73% | 年初至期末口径 |
| CET1 | 9.25% | 年初 9.34% |

宁波这份报表的增长逻辑很清晰：**不是零售突然恢复，而是对公资产快速扩张 + 手续费收入恢复得很猛。**

原文里最关键的两句话是：一是公司称 Q1 利息净收入同比 +14.44%、手续费及佣金净收入同比 +81.72%；二是贷款及垫款较年初 +9.45%、客户存款较年初 +10.89%。这说明增长是“规模 + 中收”双驱动。

但宁波也不是无成本增长。RWA 从 23,485.30 亿增至 24,797.50 亿，CET1 从 9.34% 降至 9.25%。也就是说，它是在更积极地使用资本。

| 归母净利润 | 16.82亿 | +8.23% |
| 利息净收入 | 25.65亿 | +21.73% |
| 手续费及佣金净收入 | 4.03亿 | -18.97% |

苏州银行增长好看，主要靠**资产负债表快速扩张和净利息收入高增**。但它的手续费净收入其实是下降的，投资收益也下降，信用减值损失还同比 +78.59%。

所以苏州不是“各项业务都漂亮”，而是**贷款和存款扩张太快，净利息收入把其他压力盖住了**。

这里资本消耗更明显：RWA 从 5,334.01 亿增至 5,887.26 亿，CET1 从 9.53% 降到 8.97%。这类增长对短期利润表很友好，但后续要看两件事：新投放贷款质量、资本补充能力。

杭州银行利润增速明显高于营收增速，一部分来自净利息收入高增，另一部分来自信用减值损失减少。它的手续费净收入是下降的，投资收益也大幅下降。

这意味着杭州的好看利润，不完全是收入端全面爆发，更像是：**对公贷款扩张 + 利息净收入改善 + 减值压力下降**。

## 4. 招行增长卡在哪里

### 4.1 招行不是没有增长，而是增长的“发动机”不够热

| 项目 | 招商银行 2026Q1/期末 | 变化 |
|---|---:|---:|
| 营收 | 869.40亿 | +3.81% |
| 归母净利润 | 378.52亿 | +1.52% |
| 净利息收入 | 556.42亿 | +4.99% |
| 非利息净收入 | 312.98亿 | +1.77% |
| 净手续费及佣金收入 | 206.56亿 | +4.87% |
| 财富管理手续费 | 85.07亿 | +25.42% |
| 贷款和垫款总额 | 74,643.73亿 | +2.84% |
| 公司贷款 | 34,401.73亿 | +6.98% |
| 零售贷款 | 36,830.40亿 | -1.00% |
| 客户存款 | 99,591.97亿 | +1.25% |
| 信用减值损失 | 148.46亿 | +15.65% |
| CET1 | 14.13% | 年初 14.16% |

招行的核心问题可以压缩成一句话：**最值钱的零售资产没有增长，而对公扩张虽然有，但收益率和资本回报不如零售；同时信用成本抬升把收入改善吃掉了。**

### 4.2 和头部城商行比，招行差在三处

第一，**资产扩张速度差很多。**  
招行贷款 +2.84%，城商行样本里宁波 +9.45%、苏州 +10.83%、杭州 +7.57%。这不是小差距，而是一季报利润表“好不好看”的核心差距。

第二，**招行零售贷款是负增长。**  
宁波、苏州的个人贷款也几乎没增长，杭州个人贷款甚至下降，但它们体量小、对公扩张足够猛，可以盖住零售疲软。招行过去的高质量优势来自零售金融、财富管理和低成本负债，当零售贷款 -1.00% 时，对招行的杀伤比对城商行更大。

第三，**信用成本没有帮忙。**  
招行信用减值损失 +15.65%；宁波 +21.55%、苏州 +78.59%，但它们收入端扩张足以覆盖；杭州信用减值损失反而下降 51.51%，对利润释放帮助很大。招行营收 +3.81%，信用减值 +15.65%，利润自然跑不快。

## 5. 是不是资本扩张不激进导致的？

答案是：**是表象之一，但不是根因。**

如果只看资本消耗，城商行确实更激进：

| 银行 | 贷款较年初 | RWA较年初 | CET1变化 | 判断 |
|---|---:|---:|---:|---|
| 招商银行 | +2.84% | 未在简表中直接给出同比增速 | 14.16% -> 14.13% | 资本消耗很温和 |
| 宁波银行 | +9.45% | +5.59% | 9.34% -> 9.25% | 温和消耗资本换增长 |
| 苏州银行 | +10.83% | +10.37% | 9.53% -> 8.97% | 明显消耗资本换增长 |
| 杭州银行 | +7.57% | +5.48% | 9.59% -> 9.56% | 扩张较快但资本稳定 |

但招行不是因为没资本才不扩张。招行 CET1 14.13%，远高于宁波、苏州、杭州。真正的问题是：**如果招行也学城商行那样猛扩，对应资产大概率更多落在低收益对公、票据、金融投资或区域信贷上，这会稀释招行的零售优势和 ROE，而不是自动提高长期价值。**

换句话说，招行不是缺子弹，而是好靶子变少了。

## 6. 更底层的差异：城商行扩张看起来好，招行守得更重

| 维度 | 头部城商行 | 招商银行 |
|---|---|---|
| 增长来源 | 对公/区域信贷高增，贷款和存款同步扩表 | 对公增长，零售贷款收缩 |
| 收入结构 | 净利息收入弹性更大 | 净息差高，但规模弹性不够 |
| 资本使用 | 宁波、苏州更积极消耗资本 | 资本大量富余，扩张克制 |
| 资产质量压力 | 静态指标好，但高增长后续要看迁徙 | 静态稳，但信用卡不良生成抬头 |
| 投资含义 | 短期增速更漂亮，后续要看资产质量兑现 | 短期增速慢，核心看零售资产何时恢复 |

## 7. 投资上怎么理解

招行现在的核心矛盾，不是“有没有能力扩张”，而是“是否愿意为了短期利润增速牺牲长期资产质量和资本回报”。

城商行 2026Q1 的增长好看，主要是资产负债表踩油门；招行 2026Q1 的增长慢，主要是零售资产没回来、信用成本还在消耗利润。  

所以后面看招行，不应该只盯 ROE 或利润增速，而要盯三个更早的信号：

1. **零售贷款是否止跌。**  
   只要零售贷款继续负增长，招行的高 ROE 修复就很难快。

2. **信用卡不良生成是否回落。**  
   这是信用成本能不能下降的前置信号。

3. **财富管理手续费高增能否连续 2-3 个季度。**  
   这决定非息恢复是不是一次性市场回暖，而不是招行护城河重新变现。

如果这三项改善，招行的慢增长会变成“蓄力”；如果这三项不改善，那它只是“守得很好但长不动”。
