# 东鹏饮料估值报告：DCF 与价格合理性评估

## 一、结论先行

基于当前已整理的 `2020-2025` 财务数据、`2025` 年度业绩公告、港股招股说明书以及已并入本目录的饮料可比公司资料，我把东鹏饮料重新按 `扣非口径` 做了一版估值，结论是：

- `DCF 基准价值：256.7 元/股`
- `相对估值中枢：280-305 元/股`
- `综合目标价：278 元/股`

如果用 `2026-04-10` 的最新收盘价 `200.12 元/股` 做静态对照：

- 对应上行空间约 `+38.9%`

如果只给一句话结论：

- `东鹏不属于低估烟蒂，但按扣非和最新收盘价重算后，当前价格仍低于合理价值中枢。`

## 二、这份估值报告怎么做

这次我不用单一方法拍结论，而是分三层：

1. `DCF`：给出企业内在价值
2. `相对估值`：用最像的饮料公司校验价格区间
3. `价格合理性评估`：把当前价格、管理层回购锚、分红能力、FCFF 收益率放在一起看

这里最重要的原则是：

- `DCF` 用来回答“值多少钱”
- `相对估值` 用来回答“市场通常会给这类公司什么倍数”
- `合理性评估` 用来回答“当前价格是在透支未来，还是仍有安全边际”

## 三、估值前先看业务质量：东鹏为什么值得用中高质量框架估值

### 1. 2025 年经营事实

根据 `2025` 年度业绩公告和年报：

- 营业收入：`208.66 亿元`，同比 `+31.8%`
- 归母净利润：`44.15 亿元`，同比 `+32.7%`
- 扣非归母净利润：`41.84 亿元`，同比 `+28.3%`
- 经营现金流：`61.74 亿元`
- ROE：`51.6%`

公司已经不是“单一能量饮料企业”的初级阶段，而是在向综合饮料平台进化：

- `能量饮料` 收入 `155.99 亿元`
- `运动饮料/东鹏补水啦` 收入 `32.74 亿元`
- `其他饮料` 收入 `19.86 亿元`

其中最关键的结构变化是：

- 非东鹏特饮产品收入占比从 `2024` 年的 `15.9%` 提升到 `2025` 年的 `25.2%`

如果把 `2023-2025` 连起来看，结构变化更直观：

| 年份 | 总收入 | 能量饮料占比 | 运动/电解质占比 | 其他饮料占比 |
| --- | ---: | ---: | ---: | ---: |
| 2023 | 112.6 亿元 | 91.9% | 3.5% | 4.5% |
| 2024 | 158.3 亿元 | 84.0% | 9.4% | 6.5% |
| 2025 | 208.7 亿元 | 74.8% | 15.7% | 9.5% |

也就是说，东鹏的估值不应只按“成熟单品公司”看，而应该给一部分“第二曲线正在兑现”的溢价。

### 2. 渠道和产能基础

年报披露的经营基础包括：

- 活跃终端网点超过 `450 万`
- 经销商超过 `3400 家`
- `14` 个规划生产基地中已有 `10` 个投产

这三项共同意味着：

- 东鹏的高增长，不只是品牌营销拉出来的
- 它背后有实实在在的全国渠道深度、供应链密度和规模优势

另外，`2025` 年度业绩公告里，管理层给出的 `2026` 年经营计划是：

- `2026 年营业收入增长不低于 20%`

这个目标本身就说明：

- 管理层并没有把 `2025` 视为异常高点
- 而是把 `20%+` 增长当成下一年的底线要求

### 3. 为什么这会影响估值

对饮料公司来说，决定估值上限的不是单年利润，而是三件事：

1. `单品护城河是否够深`
2. `第二曲线是否真的能长出来`
3. `现金流和再投资是否同向健康`

东鹏目前三点都不差：

- 主品类龙头地位明确
- 补水啦已经从“试验品”走到“30 亿级产品”
- 经营现金流持续高于净利润

这也是为什么我不把东鹏放到 `康师傅/统一/中国食品` 那一类成熟饮料低倍数框架里。

## 四、DCF 的底层逻辑

### 1. 为什么用 FCFF

东鹏 2025 年有两层容易误导人的特征：

- 账上有较大的金融资产和定期存款
- 同时也有不低的有息负债

所以如果只看股权口径现金流，容易被现金管理和资本结构干扰。  
这次统一用 `FCFF`，直接估企业经营资产本身的价值，再加净现金。

### 2. 2025 作为估值基年

这次我不直接拿 `44.15 亿元` 的归母净利润做估值起点，而是先看扣非：

- `2025` 归母净利润：`44.15 亿元`
- `2025` 扣非归母净利润：`41.84 亿元`
- 差额约：`2.31 亿元`

这个差额不算离谱，但已经足以影响估值中枢。  
因此这次 DCF 的基年利润，采用“按扣非差额反推并归一化经营利润”的方式处理，不再直接沿用报表原始 EBIT。

归一化后的基年输入如下：

| 项目 | 2025A |
| --- | ---: |
| 营业收入 | 208.66 亿元 |
| 归一化 EBIT | 53.93 亿元 |
| 有效税率 | 20.96% |
| 归一化税后 EBIT（NOPAT） | 42.62 亿元 |
| 折旧摊销 | 5.03 亿元 |
| 资本开支 | 22.66 亿元 |
| 经营性营运资本 | -85.84 亿元 |
| 营运资本变动 | -20.92 亿元 |
| 归一化 FCFF | 45.91 亿元 |

这里最关键的一点是：

- 东鹏的经营性营运资本长期为负

这不是坏事，反而是它渠道模式强势的体现：

- 应收很低
- 合同负债很高
- 贸易及其他应付款也能跟着规模扩张

所以东鹏是少数可以一边高增长、一边持续释放营运资金的消费公司。

## 五、DCF 预测假设

### 1. 收入增速假设

我对 `2026-2030` 的基准收入增速设为：

| 年份 | 收入增速 |
| --- | ---: |
| 2026E | 22% |
| 2027E | 18% |
| 2028E | 15% |
| 2029E | 12% |
| 2030E | 10% |

这个假设不激进，理由是：

- `2025` 已经做到了 `+31.8%`
- 管理层明确提出 `2026 年收入增长不低于 20%`
- 年报披露 `补水啦` 仍在高增长
- 全国化和多品类红利还没有走完
- 但基数上升后，增速自然要逐年回落

### 2. 利润率假设

基准情景下，我把 `归一化 EBIT Margin` 设为从 `26.0%` 缓慢抬升到 `26.5%`。

这个假设也不激进，因为：

- 东鹏的毛利率并不夸张，`2025` 为 `44.1%`
- 但它的费用效率非常强
- 全国扩张、品类扩张和产能优化仍有望继续摊薄单位成本

### 3. 再投资假设

这是这次 DCF 最重要的地方。

我没有把东鹏粗暴当成“轻资产品牌公司”，而是明确保留了较高的资本开支：

| 年份 | 折旧摊销/收入 | 资本开支/收入 | 经营性营运资本/收入 |
| --- | ---: | ---: | ---: |
| 2026E | 2.5% | 9.5% | -40.5% |
| 2027E | 2.6% | 8.8% | -40.0% |
| 2028E | 2.6% | 8.0% | -39.5% |
| 2029E | 2.6% | 7.2% | -39.0% |
| 2030E | 2.6% | 6.5% | -38.5% |

这组设定的含义是：

- 前两年继续维持较高扩产和产线投入
- 之后随着 14 基地产能逐步成形，资本开支强度缓慢回落
- 负营运资本优势仍保留，但不假设无限增强

### 4. 折现率与永续增长

基准参数：

- 无风险利率：参考 `2026-04-10` 附近中国 `10Y` 国债收益率约 `1.8%-1.9%`
- 基准 `WACC = 8.5%`
- 永续增长率 `g = 2.5%`

这组参数比上一版更谨慎，尤其是永续增长率从 `3.0%` 降到 `2.5%`，更适合作为当前估值中枢。

## 七、相对估值：东鹏该对标谁

基于已经并入本目录的 `可比公司研究`，我认为东鹏最合适的双主锚是：

1. `Monster Beverage`
2. `农夫山泉`

逻辑是：

- `Monster`：品类最像，都是功能饮料逻辑
- `农夫山泉`：国内饮料龙头里最接近“高品牌力 + 高效率 + 高 ROE + 仍在扩张”的公司

其他公司只适合作为边界参考：

- `Celsius`：代表高成长情绪上沿，但利润质量不适合作主锚
- `康师傅/统一/中国食品/KO/KDP/PEP`：更适合做成熟饮料下沿

## 八、为什么主看 PE，副看 EV/EBIT

### 1. 主看 2026E PE

因为东鹏具备几个特点：

- 净利润质量高
- 经营现金流长期强于净利润
- 税率和资本结构都不复杂到足以扭曲 PE

所以 `PE` 依然是最有效的市场语言。

### 2. 副看 EV/EBIT

东鹏账上有大量现金类资产，也有一定借款。  
所以为了避免只看 PE 忽略资产负债表差异，需要用 `EV/EBIT` 做交叉校验。

### 3. 不主看 EV/EBITDA 和 P/FCF

- `EV/EBITDA` 会把东鹏高强度扩产阶段的真实再投资成本遮掉
- `P/FCF` 会被负营运资本和 capex 周期放大波动，不适合作为单一主锚

## 九、相对估值区间

我给东鹏的相对估值区间是：

- `2026E PE：26x - 34x`
- `2026E EV/EBIT：20x - 25x`

东鹏不该掉到成熟饮料龙头那一层，但也不应直接拿到海外高情绪成长股最上沿。

### 1. PE 估值

我采用 `2026E` 扣非归母净利润 `51.04 亿元`，对应 `EPS ≈ 9.82 元`

| 2026E PE | 对应股价 |
| --- | ---: |
| 26x | 255.2 元/股 |
| 30x | 294.5 元/股 |
| 34x | 333.8 元/股 |

### 2. EV/EBIT 估值

我采用 `2026E 归一化 EBIT ≈ 66.2 亿元`，加回广义净现金 `96.5 亿元`

| 2026E EV/EBIT | 对应股价 |
| --- | ---: |
| 20x | 273.2 元/股 |
| 22.5x | 305.0 元/股 |
| 25x | 336.8 元/股 |

### 3. 相对估值结论

相对估值给出的中心带很清晰：

- 东鹏合理价值大致在 `255 - 337 元/股`
- 中枢更接近 `280 - 305 元/股`

## 十、价格合理性评估

### 1. 参考价格

为了避免用未核实的盘中价，这次我用一个已可核实的最新收盘价：

- `2026-04-10` 收盘价：`200.12 元/股`

作为静态对照。  
另外，市场上还能看到公司 `2026-04-01` 公布的 A 股回购方案，上限价格为 `248 元/股`，这提供了一个管理层视角下的价格锚。

### 2. 2025 FCFF 收益率

按 `2025A 归一化 FCFF ≈ 45.91 亿元`、参考价格 `200.12 元/股` 估算：

- 市值约 `1,041 亿元`
- 扣除广义净现金后，EV 约 `945 亿元`
- `EV / 2025 FCFF ≈ 20.6x`
- 对应 `FCFF Yield ≈ 4.9%`

这个水平对于一家：

- 仍能做到 `30%+` 收入增长
- ROE 超过 `50%`
- 负营运资本持续释放现金

的消费龙头，不算离谱。

### 3. DCF 与相对估值能否互相验证

能，而且验证得比较好：

- DCF 基准：`256.7 元`
- PE 中枢：`294.5 元`
- EV/EBIT 中枢：`305.0 元`

这三者没有出现明显撕裂。  
说明本次估值不是某一种方法单独过度乐观，而是三套方法都落在差不多的区间。

### 4. 综合目标价

我用下面的权重：

- `DCF 50%`
- `PE 30%`
- `EV/EBIT 20%`

得到综合目标价：

- `278 元/股`

## 十一、最终判断

### 1. 这只股票值不值得买

如果你的问题是：

- `东鹏是不是一只明显低估的烟蒂股`

答案是否定的。  
它不是烟蒂，也不是便宜到离谱。

如果你的问题是：

- `东鹏当前价格是否已经把未来几年价值全部透支`

我的答案也是否定的。

### 2. 我怎么看当前价格

东鹏当前更像：

- 一只 `高质量、高成长、高现金流效率` 的消费成长股
- 市场已经给了它溢价
- 但这个溢价还没有高到完全失去性价比

### 3. 最后的估值结论

- `保守价值区间：238 - 255 元/股`
- `合理价值区间：257 - 305 元/股`
- `乐观价值区间：330 元/股以上`

所以这次我的最终给法是：

- `目标价：278 元/股`
- `评级口径：偏积极，但不是无脑高估`
