# 霸王茶姬 2025Q4 分红与商誉复核

## 结论先行

这次复核后，可以比较稳地确认三件事：

1. `2025Q4 融资现金流流出 12.01 亿元`，高概率主要就是 `2025Q3 宣布、2025Q4 支付` 的特别分红。
2. `2025Q4 商誉增加 8614.8 万元`，高概率对应 `国内加盟店转直营` 的业务收购，而不是普通新开店。
3. 我前一版把 `Q3->Q4 固定资产、库存等增量` 过度归因给了 `加盟转直营`，这个推断不稳，需要推翻。更合理的结论是：
   - 可以推断 `转直营门店数`
   - 可以推断 `单店商誉`
   - 但不能仅凭报表增量，直接推出 `单店总收购价 = 130-170 万`

## 一、分红复核：Q4 的 12 亿融资流出，大概率就是特别分红

### 官方已披露信息

- 公司在 `2025-11-28` 宣布特别现金股息，总额约 `1.77 亿美元`
- 支付时间为 `2025-12-15` 左右，落在 `2025Q4`
- `2025Q4` 现金流量表中，融资活动现金流净流出为 `12.0144 亿元人民币`

### 对比表

| 项目 | 数值 |
| --- | ---: |
| 特别股息总额 | 1.77 亿美元 |
| 按 6.8 汇率折算 | 12.04 亿元人民币 |
| 2025Q4 融资现金流净流出 | 12.01 亿元人民币 |

### 结论

这两个数几乎完全对上，因此：

- `Q4 的 -12.01 亿元融资流出`，高概率就是 `特别分红支付`
- 不需要再优先怀疑成别的抽象资本项

## 二、门店结构复核：商誉只能主要对应国内加盟转直营

### Q3 到 Q4 门店变化

| 门店类型 | 2025Q3 | 2025Q4 | 变化 |
| --- | ---: | ---: | ---: |
| 总门店 | 7,338 | 7,453 | +115 |
| 加盟店 | 6,971 | 6,838 | -133 |
| 直营店 | 367 | 615 | +248 |
| 国内加盟 | 6,836 | 6,700 | -136 |
| 国内直营 | 240 | 408 | +168 |
| 海外加盟 | 135 | 138 | +3 |
| 海外直营 | 127 | 207 | +80 |

### 结论

这组变化说明：

- `海外` 这一段更像 `新增直营店`，因为海外加盟店没有减少，反而略增
- `国内` 这一段明显存在 `加盟转直营`

因此，`Q4 国内加盟转直营门店数` 的合理区间是：

- 下限：`136 家`
- 上限近似：`168 家`
- 研究中枢：`约 150 家`

## 三、商誉复核：能确认的是单店商誉，不是单店总收购价

### 商誉变化

| 项目 | 2025Q3 | 2025Q4 | 增量 |
| --- | ---: | ---: | ---: |
| 商誉 | 1,183.7 万元 | 9,798.5 万元 | 8,614.8 万元 |

### 对应到转直营门店数

| 假设转直营门店数 | 单店商誉 |
| ---: | ---: |
| 136 家 | 63.3 万元/店 |
| 150 家 | 57.4 万元/店 |
| 168 家 | 51.3 万元/店 |

### 结论

这部分能比较稳地说：

- `单店形成的商誉` 大约在 `51 万到 63 万`

但这里要强调：

- `商誉` 只是 `收购价 - 可辨认净资产公允价值`
- 它不是总收购价本身

所以，单靠 `8614.8 万商誉`，最多只能确认：

- 公司确实为一批门店业务支付了溢价
- 但不能直接确认总收购价到底是多少

## 四、为什么前一版“130-170 万/店”的结论不稳

前一版的问题在于：

- 把 `Q3->Q4 固定资产、库存、使用权资产` 的增量，过度归因给了 `加盟转直营`

这个归因不成立，因为这些增量里还混着：

- 海外直营新增 `80 家`
- 国内直营净新增里，除转换之外的新增部分
- 总部和系统层面的正常资本开支
- 新增直营店带来的租赁资产和租赁负债同步确认

### 用 Q2->Q3 做基准样本

`Q2->Q3` 这段的特点是：

- 商誉 `没有变化`
- 更接近 `正常新增直营店` 的资产增量样本

#### Q2->Q3

| 项目 | 变化 |
| --- | ---: |
| 直营店净增 | +128 家 |
| 固定资产净增 | +7,947.7 万元 |
| 存货净增 | +2,564.2 万元 |
| 固定资产+存货净增 | +1.051 亿元 |
| 对应单店固定资产+存货增量 | 82.1 万元/店 |
| 商誉增量 | 0 |

#### Q3->Q4

| 项目 | 变化 |
| --- | ---: |
| 直营店净增 | +248 家 |
| 固定资产净增 | +1.644 亿元 |
| 存货净增 | +5,489.5 万元 |
| 固定资产+存货净增 | +2.193 亿元 |
| 对应单店固定资产+存货增量 | 88.4 万元/店 |
| 商誉增量 | +8,614.8 万元 |

### 对比后的关键结论

`Q2->Q3` 在 `没有商誉变化` 的情况下，单店固定资产+存货增量已经达到 `82.1 万元/店`。  
而 `Q3->Q4` 的对应数是 `88.4 万元/店`，并没有离谱跳升。

这说明：

- `Q3->Q4` 的大部分固定资产和库存增量，本身就可以由 `正常新增直营店` 解释
- 不能把这些增量大笔算进 `加盟转直营收购价`

所以，前一版用这些资产增量去反推 `130-170 万/店`，证据不够硬。

## 五、再用现金流做一层约束：大额现金并购并不明显

### Q4 投资现金流

| 项目 | 数值 |
| --- | ---: |
| 2025Q4 投资活动现金流净流出 | 3.202 亿元 |
| 其中短期投资新增 | 1.001 亿元 |
| 剩余可用于 capex / 并购 / 其他投资 | 约 2.201 亿元 |

### 用 Q2->Q3 的正常开店样本做对照

`Q2->Q3` 的正常新增直营样本显示：

- 单店固定资产+存货增量约 `82.1 万元/店`

如果把这个样本粗略套到 `Q3->Q4` 新增的 `248 家直营店`：

- 仅正常新增直营，就可能需要约 `2.04 亿元` 的固定资产+存货投入

而 `Q4` 在扣除新增短期投资后，投资现金流剩余也只有 `2.20 亿元` 左右。

### 结论

这说明：

- `Q4` 并没有留下很大的现金空间，去支持一笔“明显偏贵的大额现金回收加盟店”
- 更合理的解释是：
  - 大部分投资现金流仍然是正常新增直营和其他 capex
  - 加盟转直营的收购对价，未必主要体现为大额现金支付
  - 其中一部分更可能通过 `承接租约、承接门店经营资产负债、支付有限溢价` 的方式完成

## 六、这次复核后，能稳地说什么，不能稳地说什么

### 能稳地说

1. `Q4 融资现金流 -12.01 亿元`，高概率主要就是 `Q3 宣布的特别分红`
2. `Q4 商誉 +8614.8 万元`，高概率主要对应 `国内加盟转直营`
3. `Q4 国内加盟转直营门店数` 大概率在 `136 到 168 家`
4. `单店商誉` 大约在 `51 万到 63 万`

### 不能稳地说

1. 不能只用报表增量，直接确认 `单店总收购价`
2. 不能只用固定资产、库存、使用权资产的变动，就反推出 `单店净资产`
3. 不能在没有 business combination 附注的情况下，精确确认：
   - 单店现金对价
   - 单店可辨认净资产公允价值
   - 单店并购溢价率

## 七、当前更合理的判断

这次复核后，更合理的表述应该是：

- 我们目前最稳的量，不是 `单店总收购价`
- 而是：
  - `转直营门店数 = 136 到 168 家`
  - `单店商誉 = 51 到 63 万`

至于 `单店总收购价`：

- 现有报表信息不够
- 前一版给出的 `130-170 万/店` 证据不够硬
- 更稳妥的做法，是等后续 `20-F` 或更完整附注出来，再确认 business combination 的对价分摊
