# 霸王茶姬 2025Q4 分红与商誉复核

## 结论先行

这次复核后，可以比较稳地确认三件事：

1. `2025Q4 融资现金流流出 12.01 亿元`，高概率主要就是 `2025Q3 宣布、2025Q4 支付` 的特别分红。
2. `2025Q4 商誉增加 8614.8 万元`，高概率对应 `国内加盟店转直营` 的业务收购，而不是普通新开店。
3. 我前一版把 `Q3->Q4 固定资产、库存等增量` 过度归因给了 `加盟转直营`，这个推断不稳，需要推翻。更合理的结论是：
   - 可以推断 `转直营门店数`
   - 可以推断 `单店商誉`
   - 但不能仅凭报表增量，直接推出 `单店总收购价 = 130-170 万`

## 一、分红复核：Q4 的 12 亿融资流出，大概率就是特别分红

### 官方已披露信息

- 公司在 `2025-11-28` 宣布特别现金股息，总额约 `1.77 亿美元`
- 支付时间为 `2025-12-15` 左右，落在 `2025Q4`
- `2025Q4` 现金流量表中，融资活动现金流净流出为 `12.0144 亿元人民币`

### 对比表

| 项目 | 数值 |
| --- | ---: |
| 特别股息总额 | 1.77 亿美元 |
| 按 6.8 汇率折算 | 12.04 亿元人民币 |
| 2025Q4 融资现金流净流出 | 12.01 亿元人民币 |

### 结论

这两个数几乎完全对上，因此：

- `Q4 的 -12.01 亿元融资流出`，高概率就是 `特别分红支付`
- 不需要再优先怀疑成别的抽象资本项

## 二、门店结构复核：商誉只能主要对应国内加盟转直营

### Q3 到 Q4 门店变化

| 门店类型 | 2025Q3 | 2025Q4 | 变化 |
| --- | ---: | ---: | ---: |
| 总门店 | 7,338 | 7,453 | +115 |
| 加盟店 | 6,971 | 6,838 | -133 |
| 直营店 | 367 | 615 | +248 |
| 国内加盟 | 6,836 | 6,700 | -136 |
| 国内直营 | 240 | 408 | +168 |
| 海外加盟 | 135 | 138 | +3 |
| 海外直营 | 127 | 207 | +80 |

### 结论

这组变化说明：

- `海外` 这一段更像 `新增直营店`，因为海外加盟店没有减少，反而略增
- `国内` 这一段明显存在 `加盟转直营`

因此，`Q4 国内加盟转直营门店数` 的合理区间是：

- 下限：`136 家`
- 上限近似：`168 家`
- 研究中枢：`约 150 家`

## 三、商誉复核：能确认的是单店商誉，不是单店总收购价

### 商誉变化

| 项目 | 2025Q3 | 2025Q4 | 增量 |
| --- | ---: | ---: | ---: |
| 商誉 | 1,183.7 万元 | 9,798.5 万元 | 8,614.8 万元 |

### 对应到转直营门店数

| 假设转直营门店数 | 单店商誉 |
| ---: | ---: |
| 136 家 | 63.3 万元/店 |
| 150 家 | 57.4 万元/店 |
| 168 家 | 51.3 万元/店 |

### 结论

这部分能比较稳地说：

- `单店形成的商誉` 大约在 `51 万到 63 万`

但这里要强调：

- `商誉` 只是 `收购价 - 可辨认净资产公允价值`
- 它不是总收购价本身

所以，单靠 `8614.8 万商誉`，最多只能确认：

- 公司确实为一批门店业务支付了溢价
- 但不能直接确认总收购价到底是多少

## 四、为什么前一版“130-170 万/店”的结论不稳

前一版的问题在于：

- 把 `Q3->Q4 固定资产、库存、使用权资产` 的增量，过度归因给了 `加盟转直营`

这个归因不成立，因为这些增量里还混着：

- 海外直营新增 `80 家`
- 国内直营净新增里，除转换之外的新增部分
- 总部和系统层面的正常资本开支
- 新增直营店带来的租赁资产和租赁负债同步确认

### 用 Q2->Q3 做基准样本

`Q2->Q3` 这段的特点是：

- 商誉 `没有变化`
- 更接近 `正常新增直营店` 的资产增量样本

#### Q2->Q3

| 项目 | 变化 |
| --- | ---: |
| 直营店净增 | +128 家 |
| 固定资产净增 | +7,947.7 万元 |
| 存货净增 | +2,564.2 万元 |
| 固定资产+存货净增 | +1.051 亿元 |
| 对应单店固定资产+存货增量 | 82.1 万元/店 |
| 商誉增量 | 0 |

#### Q3->Q4

| 项目 | 变化 |
| --- | ---: |
| 直营店净增 | +248 家 |
| 固定资产净增 | +1.644 亿元 |
| 存货净增 | +5,489.5 万元 |
| 固定资产+存货净增 | +2.193 亿元 |
| 对应单店固定资产+存货增量 | 88.4 万元/店 |
| 商誉增量 | +8,614.8 万元 |

### 对比后的关键结论

`Q2->Q3` 在 `没有商誉变化` 的情况下，单店固定资产+存货增量已经达到 `82.1 万元/店`。  
而 `Q3->Q4` 的对应数是 `88.4 万元/店`，并没有离谱跳升。

这说明：

- `Q3->Q4` 的大部分固定资产和库存增量，本身就可以由 `正常新增直营店` 解释
- 不能把这些增量大笔算进 `加盟转直营收购价`

所以，前一版用这些资产增量去反推 `130-170 万/店`，证据不够硬。

## 五、按“248 家直营增量 - 转直营门店数”重算一次

用户提出的修正是对的：

- `Q3->Q4` 直营店净增 `248 家`
- 其中并不是全部都应该按“正常新开直营店”处理
- 更合理的做法是：
  - 先估 `转直营门店数`
  - 再用 `248 - 转直营门店数` 去估正常新开直营店的资产增量
  - 剩余部分，再去看是否可能与加盟转直营相关

### 第一步：用 Q2->Q3 样本得到“正常新开直营店”的资产强度

`Q2->Q3`：

- 直营店净增：`128 家`
- 固定资产+存货增量：`1.051 亿元`

对应：

- 单店固定资产+存货增量约 `82.1 万元/店`

### 第二步：代入 Q3->Q4 的转直营门店假设

| 假设转直营门店数 | 推测正常新开直营店数 | 正常新开店对应固定资产+存货 | 剩余固定资产+存货 |
| ---: | ---: | ---: | ---: |
| 136 家 | 112 家 | 0.920 亿元 | 1.273 亿元 |
| 150 家 | 98 家 | 0.805 亿元 | 1.388 亿元 |
| 168 家 | 80 家 | 0.657 亿元 | 1.536 亿元 |

把这个“剩余固定资产+存货”再除以转直营门店数，对应为：

| 假设转直营门店数 | 单店剩余固定资产+存货 | 单店商誉 | 单店合计 |
| ---: | ---: | ---: | ---: |
| 136 家 | 93.6 万元 | 63.3 万元 | 157.0 万元 |
| 150 家 | 92.5 万元 | 57.4 万元 | 150.0 万元 |
| 168 家 | 91.4 万元 | 51.3 万元 | 142.7 万元 |

### 这一步该怎么理解

这说明：

- 用户提出的修正方向是对的
- 我前一版直接拿 `248 家 x 82.1 万` 去顶满正常开店，表述过头了
- 按这次修正后，`Q3->Q4` 的确给加盟转直营留出了一块更明显的“剩余资产空间”

但是这里仍然要分清楚：

- 这块 `91-94 万/店` 的数，代表的是“扣除正常新开直营后，可能与转直营相关的固定资产+存货”
- 它不是严格意义上的 `收购 cash price`
- 它也不是严格意义上的 `可辨认净资产`

原因是：

- 它只看了 `固定资产 + 存货`
- 没有纳入其他流动资产
- 也没有扣掉应付、应计、租赁等负债
- 更没有区分哪些项目是 `业务合并并表`，哪些只是 `后续补装修和补设备`

因此，这一步更合理的解释是：

- 它给出了一个更接近现实的 `资产桥`
- 但它还不能直接被命名为 `单店收购价`

## 六、这次复核后，能稳地说什么，不能稳地说什么

## 六、把使用权资产和租赁负债单独拆开：这一步对“现金付价”最关键

如果目标是判断：

- 公司到底为了收加盟店，`真金白银付了多少现金`
- 有没有明显高价回收、损害股东利益

那 `使用权资产` 这一项不能单独看，必须和 `租赁负债` 一起看。

### Q3->Q4 租赁相关变化

| 项目 | Q3->Q4 变化 |
| --- | ---: |
| 使用权资产 | +2.808 亿元 |
| 流动租赁负债 | +1.084 亿元 |
| 非流动租赁负债 | +1.647 亿元 |
| 租赁负债合计 | +2.731 亿元 |
| 净租赁资产变化 | +0.077 亿元 |

### 结论

这说明：

- `Q4` 增加的使用权资产，几乎被新增租赁负债一一对冲
- 也就是说，`房租相关资产` 大部分不是现金买出来的，而是随着直营门店并表和租约承接一起进表

如果把这 `0.077 亿元` 的净租赁资产摊到 `136 到 168 家` 转直营门店：

- 单店净租赁资产大概只有 `4.6 万到 5.7 万`

因此：

- `143-157 万/店` 这个区间，本身并没有把一大块“房租资产”硬塞进去
- 就算把租赁净额加进去，单店量级也只会再抬一点点，不会根本改写结论

## 七、进一步逼近“现金到底付了多少”：用投资现金流做上限约束

如果关注的是 `现金是否被高价抽走`，那比起看总资产，更应该看：

- `Q4 投资活动现金流`
- 扣掉和收购无关的项目之后，还剩多少

### Q4 投资现金流拆分

| 项目 | 数值 |
| --- | ---: |
| 2025Q4 投资活动现金流净流出 | 3.202 亿元 |
| 其中新增短期投资 | 1.001 亿元 |
| 扣除短期投资后的剩余投资现金流 | 2.201 亿元 |

### 用 Q2->Q3 的“正常新增直营”作为现金基准样本

`Q2->Q3` 的特点是：

- 商誉不变
- 更接近正常新增直营和正常总部 capex

当季：

- 投资现金流净流出：`1.955 亿元`
- 直营店净增：`128 家`

对应的研究口径强度约为：

- `152.7 万/每家新增直营店`

这个数不是“单店装修成本”，而是：

- 当季全部投资现金流
- 除以新增直营店数量
- 所以它本身已经偏保守，适合作为“现金上限基准”

### 代入 Q3->Q4

| 假设转直营门店数 | 推测正常新开直营店数 | 按 Q2->Q3 基准推测正常新增直营需占用投资现金流 | Q4 剩余投资现金流上限 | 折算单店上限 |
| ---: | ---: | ---: | ---: | ---: |
| 136 家 | 112 家 | 1.711 亿元 | 0.491 亿元 | 36.1 万/店 |
| 150 家 | 98 家 | 1.497 亿元 | 0.705 亿元 | 47.0 万/店 |
| 168 家 | 80 家 | 1.222 亿元 | 0.979 亿元 | 58.3 万/店 |

### 这张表该怎么理解

这张表不是在证明：

- 茶姬一定只付了 `36 到 58 万/店`

而是在说明：

- 按现有公开报表，`Q4 能被解释为转直营额外现金流出的空间`，其实没有想象中那么大
- 即使把剩余投资现金流全部算给转直营，这个“现金上限”也大约在 `36 到 58 万/店`

而现实里，这个数还很可能要再往下调，因为剩余投资现金流里还可能混着：

- 总部设备和系统投入
- 其他非门店类资本开支
- 新店补设备、补装修、保证金等

所以，如果你的核心问题是：

- `茶姬是不是为了回收加盟店，真金白银付了很高的现金对价`

那当前公开数据给出的信号是：

- `没有看到支持“大额现金高价回收”的证据`
- 真正比较明显的是：
  - `门店业务被并表`
  - `租约和租赁负债一起进表`
  - `形成了 51 到 63 万/店的商誉`

## 八、这次复核后，能稳地说什么，不能稳地说什么

### 能稳地说

1. `Q4 融资现金流 -12.01 亿元`，高概率主要就是 `Q3 宣布的特别分红`
2. `Q4 商誉 +8614.8 万元`，高概率主要对应 `国内加盟转直营`
3. `Q4 国内加盟转直营门店数` 大概率在 `136 到 168 家`
4. `单店商誉` 大约在 `51 万到 63 万`
5. `使用权资产` 大部分被 `租赁负债` 对冲，不应被直接当成现金付价
6. 以公开报表做保守约束，`Q4 转直营对应的额外现金流出上限` 大约在 `36 到 58 万/店`

### 不能稳地说

1. 不能只用报表增量，直接确认 `单店总收购价`
2. 不能只用固定资产、库存、使用权资产的变动，就反推出 `单店净资产`
3. 不能在没有 business combination 附注的情况下，精确确认：
   - 单店现金对价
   - 单店可辨认净资产公允价值
   - 单店并购溢价率

## 九、当前更合理的判断

这次复核后，更合理的表述应该是：

- 我们目前最稳的量，不是 `单店总收购价`
- 而是：
  - `转直营门店数 = 136 到 168 家`
  - `单店商誉 = 51 到 63 万`
  - `单店额外现金流出上限 = 36 到 58 万`

至于 `单店总收购价`：

- 如果只做粗桥接，按这次修正后的算法，`固定资产+存货+商誉` 的组合量级，大约在 `143-157 万/店`
- 但这个数仍然不能直接等同于 `收购对价`
- 它更像是“被并表门店带进来的部分资产，加上一笔溢价”的量级参考
- 更稳妥的做法，是等后续 `20-F` 或更完整附注出来，再确认 business combination 的对价分摊
