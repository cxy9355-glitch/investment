# 2026全年利润与目标价

更新时间：2026-03-30

## 1. 先说结论

我当前对霸王茶姬的判断是：

- `2026E 研究口径利润` 中枢约 `21.5 亿元人民币`
- 对应当前股价，市场大约只给了 `5.8x` 左右的 `2026E PE`
- 这个估值明显偏低

我的目标价判断是：

- `目标价：30.0 美元 / ADS`

对应：

- `18x 2026E PE`

以 `2026-03-27` 收盘价 `9.53 美元 / ADS` 为基准，对应上行空间约：

- `+215%`

## 2. 当前市场锚点

当前股价我采用的是最新可见收盘价：

- `CHA = 9.53 美元 / ADS`
- 时间：`2026-03-27` 收盘

当前股本我采用的是：

- `185.75m` 股 / ADS

这两个数都来自当前公开行情页，可作为当前估值锚：

- 股价：[StockAnalysis - CHA Statistics](https://stockanalysis.com/stocks/cha/statistics/)
- 该页同时给出了 `Shares Outstanding = 185.75M`

据此，对应当前市值约：

- `17.7 亿美元`
- 折合约 `123 亿元人民币`（按 `USD/CNY = 6.95` 的研究汇率）

## 3. 2026E 全年利润怎么外推

### 3.1 Q1 利润锚点

我在主文档里已经把 `2026Q1` 收敛到了两组区间：

- 偏保守基准：`5.0 亿元`
- 财报中性中枢：`5.1 亿 ~ 5.2 亿元`

对应文档：

- [04_GMV和利润推导过程.md](E:/Codex/investment/霸王茶姬研究/04_GMV和利润推导过程.md)

### 3.2 为什么不用 2025Q2 / Q3 做全年利润季节性模板

我没有直接拿 `2025Q2` 和 `2025Q3` 的净利润去推全年，原因很简单：

- `2025Q2` 明显被 `IPO 触发的股份支付` 扭曲
- `2025Q3` 虽然比 `Q2` 好，但仍然夹杂较重的竞争扰动和口径噪音

所以这次更稳的做法是：

- 用 `2024` 的季度净利润季节性来给 `2026` 做全年外推

### 3.3 2024 的利润季节性

根据 [01_经营总表.md](E:/Codex/investment/霸王茶姬研究/01_经营总表.md)：

- `2024Q1 净利润 = 5.951 亿元`
- `2024 全年净利润 = 25.146 亿元`

所以：

- `2024Q1 / 2024FY = 23.7%`

这个比例的含义是：

- 在一个相对正常、没有被大额一次性项目严重扭曲的年份里
- `Q1` 大约占全年利润的 `23% ~ 24%`

### 3.4 用 Q1 外推 2026 全年利润

因此：

- 如果 `2026Q1 = 4.99 亿元`
- 那么 `2026FY ≈ 4.99 / 23.7% = 21.1 亿元`

- 如果 `2026Q1 = 5.15 亿元`
- 那么 `2026FY ≈ 5.15 / 23.7% = 21.7 亿元`

所以我给 `2026E` 的利润区间是：

- `保守：21.1 亿元`
- `中性：21.5 亿元`
- `偏乐观：22.0 亿元`

## 4. 为什么估值我用 PE，不用 DCF

这家公司现在最麻烦的地方，不是算不出利润，而是：

- 新模式下 `收入确认口径` 还在变化
- 加盟端的 `17% + 折扣池偏差` 本质是研究口径利润贡献
- 所以如果现在强行上 DCF，很容易把中间口径做假精确

因此这次更稳的估值方法是：

- 先用 `2026E PE` 做目标价

原因是：

- 用户真正关心的是当前股价贵不贵
- `利润` 比 `收入` 现在更容易收敛
- 当前股价本身已经给了一个非常低的隐含 `2026E PE`

## 5. 目标价怎么给

### 5.1 每股盈利

按 `185.75m` 股 / ADS 的股本口径：

如果 `2026E 利润 = 21.5 亿元人民币`，那么：

- `2026E EPS ≈ 11.57 元人民币`
- 按 `USD/CNY = 6.95`
- `2026E EPS ≈ 1.67 美元 / ADS`

### 5.2 给多少倍 PE 合理

我不愿意给太激进的倍数，原因是：

- 现制茶行业竞争还很激烈
- 新模式还在验证
- 同店压力没有完全过去

但我也不愿意给太低的倍数，原因是：

- 霸王茶姬不是低质资产
- 品牌心智、门店模型、利润率、海外可选性都比一般茶饮公司更强
- 当前股价已经跌到了极低的隐含估值

所以我给：

- `15x PE` 作为保守估值
- `18x PE` 作为基准估值
- `20x PE` 作为乐观估值

### 5.3 对应目标价

| 情景 | 2026E 利润（亿元人民币） | 2026E PE | 对应目标价（美元 / ADS） |
| --- | ---: | ---: | ---: |
| 保守 | 21.1 | 15x | 24.5 |
| 基准 | 21.5 | 18x | 30.0 |
| 乐观 | 22.0 | 20x | 34.1 |

## 6. 当前股价怎么看

如果当前股价按 `9.53 美元 / ADS` 算：

- 对应 `2026E 利润 21.5 亿元人民币`
- 当前隐含 `2026E PE` 大约只有 `5.8x`

这是一个非常低的估值。

更直白地说：

- 市场现在的定价，已经接近把霸王茶姬当成一个“增长明显放缓、模式要打折、估值要大幅压缩”的资产在看

而如果你认为：

- 新模式不是失效，只是仍在磨合
- Q1 利润大概率还能站住 `5 亿+`
- 全年利润仍有 `21.5 亿` 左右

那么这个价格就是明显低估。

## 7. 最终判断

我的最终判断是：

- `当前股价低估`
- `目标价 = 30.0 美元 / ADS`

这个目标价对应的是：

- `2026E 利润 21.5 亿元`
- `18x PE`

这是一个并不激进的估值框架。  
如果后面你把：

- `折扣池使用率`
- `总部费用率`

继续收紧，而结果仍然支持 `2026E 利润在 21.5 亿左右`，那这套目标价还有上修空间。

## 8. 主要风险

这份目标价最容易错在 4 个地方：

1. `折扣池实际使用率` 比我现在假设的更高  
这会压低加盟端利润贡献

2. `总部费用率` 长时间停在 `11.8%` 以上  
这会把全年利润压到 `21 亿` 以下

3. `2026Q2-Q4` 的同店表现比 Q1 更差  
那 `21.5 亿` 的全年利润中枢就要下修

4. 市场不愿意给 `18x PE`  
如果市场只愿意给 `15x`，那目标价就会落到 `25 美元` 左右
