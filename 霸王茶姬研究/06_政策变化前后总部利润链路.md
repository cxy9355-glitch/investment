# 政策变化前后总部利润链路

更新时间：2026-03-30

## 1. 总体结构

后面统一不再强行写成一个“利润率公式”，而是只保留利润链路：

`总部利润 = 加盟端利润 + 直营端门店经营利润 - 总部费用`

也就是分三段看：

1. `加盟端利润`
2. `直营端门店经营利润`
3. `减：总部费用`

## 2. 旧模式

### 2.1 加盟端

旧模式下，加盟端的链路是：

`加盟GMV`
`-> 卖货收入 + 服务费收入`
`-> 扣卖货对应的采购、仓储、物流成本`
`-> 加盟端利润`

换句话说，旧模式加盟端利润的本质来源是：

1. `卖货毛利`
2. `服务费`

### 2.2 直营端

直营端统一按下面顺序看：

`直营GMV`
`-> 实收`
`-> 原材料成本`
`-> 门店毛利收入`
`-> 扣房租、水电、人工和其他门店经营成本`
`-> 直营端门店经营利润`

其中：

1. `实收 = 直营GMV × 实收率`
2. `原材料成本 = 直营GMV × (1 - 毛利率)`
3. `门店毛利收入 = 实收 - 原材料成本`

### 2.3 总部费用

总部费用单独看，不放进加盟端或直营端里混算：

`总部费用 = 其他运营成本 + 销售费用 + 管理费用`

### 2.4 旧模式总部利润链路

所以旧模式下，总部利润链路就是：

`加盟GMV`
`-> 加盟端利润`

加上

`直营GMV`
`-> 直营端门店经营利润`

再减：

`总部费用`

得到：

`旧模式总部利润`

## 3. 新模式

### 3.1 加盟端

新模式下，加盟端不再按“卖货毛利”作为核心，而是按“服务费 + 折扣池偏差”来理解。

链路写成：

`加盟GMV`
`-> 17%服务费收入`
`-> 加上或减去折扣池偏差`
`-> 加盟端利润`

其中：

`折扣池偏差 = 10%折扣池 - 实际折扣使用`

当前工作口径再补一句：

1. 如果实际折扣使用 `<=10%`
未用完部分留在总部收益里

2. 如果实际折扣使用 `>10%`
超出部分再额外冲减总部利润

### 3.2 物料与仓储

按当前已经确认的口径：

1. 物料、仓储、履约都已经体现在加盟商进货价里
2. 总部在物料这里 `既不赚钱，也不亏钱`

所以在这版链路里：

`物料与仓储不再单独作为总部利润来源，也不单独作为总部利润扣减项`

### 3.3 直营端

直营端链路不变，仍然是：

`直营GMV`
`-> 实收`
`-> 原材料成本`
`-> 门店毛利收入`
`-> 扣房租、水电、人工和其他门店经营成本`
`-> 直营端门店经营利润`

### 3.4 总部费用

总部费用口径也不变，仍然是：

`总部费用 = 其他运营成本 + 销售费用 + 管理费用`

### 3.5 新模式总部利润链路

所以新模式下，总部利润链路就是：

`加盟GMV`
`-> 17%服务费收入`
`-> 折扣池偏差`
`-> 加盟端利润`

加上

`直营GMV`
`-> 直营端门店经营利润`

再减：

`总部费用`

得到：

`新模式总部利润`

## 4. 新旧模式真正要比较的东西

政策变化前后，真正要比较的不是把所有东西强行凑成一个“利润率”。

真正要比较的是这三段链路在新旧模式下分别怎么变：

1. `加盟端利润` 怎么变
2. `直营端门店经营利润` 变不变
3. `总部费用` 怎么变

所以最后的对比应该写成：

旧模式：

`总部利润 = 旧加盟端利润 + 直营端门店经营利润 - 总部费用`

新模式：

`总部利润 = 新加盟端利润 + 直营端门店经营利润 - 总部费用`

## 5. 当前版本的结论

这版文档只保留链路，不再强行输出一个统一分母下的“总部利润率”。

后面如果继续往数值推，应该按顺序分别确定：

1. `旧加盟端利润`
2. `新加盟端利润`
3. `直营端门店经营利润`
4. `总部费用`

最后再合成总部利润。
