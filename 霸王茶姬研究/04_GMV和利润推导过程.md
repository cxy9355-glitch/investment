# GMV和利润推导过程

更新时间：2026-03-30

## 1. 本文档用途

从现在开始，`霸王茶姬` 这条线只继续维护这一张主文档。

这张表只做一件事：

- 用同一条利润链，直接给出 `2025Q1 实际` 和 `2026Q1 测算` 的可对比结果

本文里的“利润”，统一指与 `2025Q1` 官方净利润尽量对齐的研究口径近似值。

## 2. 当前结论

如果严格按 [06_政策变化前后总部利润链路.md](E:/Codex/investment/霸王茶姬研究/06_政策变化前后总部利润链路.md) 的 `新模式` 去算，并且把 `折扣池实际使用率 = 10%` 当作基准，也就是“折扣池基本用满”，那么：

- `2026Q1 利润 ≈ 5.4 亿元`

和 `2025Q1` 官方净利润 `6.8 亿元` 相比：

- `同比约 -20.9%`

这版结论的关键点是：

- `GMV` 没有明显塌掉
- `直营扩张` 确实带来了利润增量
- 但在 `新模式` 下，如果 `10% 折扣池` 基本被用满，加盟端利润并不会明显比旧模式更高
- 再叠加更高的总部费用，最终 `26Q1` 利润仍低于 `25Q1`

## 3. 统一利润链

后面统一按这一条式子推：

`利润 = 加盟端利润 + 直营端门店经营利润 - 总部费用`

其中：

- `加盟端利润`：本次严格按 `新模式` 链条测算
- `直营端门店经营利润`：按 `直营GMV -> 实收 -> 原材料成本 -> 门店经营成本 -> 利润` 测算
- `总部费用`：`其他运营成本 + 销售费用 + 管理费用`

`677.3 + 824.6 - 85.8 = 1,416.1`

| --- | ---: | --- |

| 利润 | 677.3 | 官方净利润 |

## 5. 先用 25Q3 和 26Q1 做线性插值，得到 25Q4

`2025Q4 期末门店数 = (2025Q3 期末门店数 + 2026Q1 期末门店数) / 2`

再进一步：

`2026Q1 平均营业门店数 = (2025Q4 期末门店数 + 2026Q1 期末门店数) / 2`

## 7. 2026Q1 利润详细推导

### 7.1 加盟端利润：严格按新模式链条

这次不再直接拍一个“加盟端利润率”，而是直接按 `文档 6` 的新模式去走：

`加盟GMV -> 17%服务费收入 -> 折扣池偏差 -> 加盟端利润`

其中：

`折扣池偏差 = 10%折扣池 - 实际折扣使用率`

基准情景下，我先取：

- `实际折扣使用率 = 10%`

也就是：

- `折扣池基本用满`
- `折扣池偏差 = 0`

因此基准情景下的加盟端利润率就是：

- `17%`

所以：

`加盟端利润 = 7,394.1 × 17% = 1,257.0`

即：

- `2026Q1 加盟端利润 = 1,257.0 百万元`

### 7.2 直营端门店经营利润

直营端继续按你前面确认的更合理口径来推：

- `直营实收率 = 80%`
- `直营毛利率 = 62%`
- `直营门店经营成本率 = 15.5%`

对应链条：

| 项目 | 2026Q1 基准（百万元） | 计算方式 |
| --- | ---: | --- |
| 直营GMV | 860.5 | 上一步测算 |
| 实收 | 688.4 | `860.5 × 80%` |
| 原材料成本 | 327.0 | `860.5 × (1 - 62%)` |
| 门店毛利收入 | 361.4 | `688.4 - 327.0` |
| 直营门店经营成本 | 133.4 | `860.5 × 15.5%` |
| 直营端门店经营利润 | 228.0 | `361.4 - 133.4` |

所以：

- `2026Q1 直营端门店经营利润 = 228.0 百万元`

### 7.3 总部费用

总部费用这一步单独看，不压回加盟端，也不压回直营端。

基准假设：

- `总部费用率 = 11.5%`

所以：

`总部费用 = 8,254.6 × 11.5% = 949.3`

即：

- `2026Q1 总部费用 = 949.3 百万元`

### 7.4 最终利润合成

把三段合起来：

`利润 = 1,257.0 + 228.0 - 949.3 = 535.7`

即：

- `2026Q1 利润 = 535.7 百万元`
- `约 5.4 亿元`

- 但 `总部费用` 也同步抬升
- 所以最后利润还是低于 `25Q1`

## 9. 折扣池使用率敏感性

因为新模式下加盟端利润直接取决于：

`17% + (10% - 实际折扣使用率)`

所以只要 `折扣池` 没有被完全用满，利润会显著上移。

在其他参数不变时：

| 实际折扣使用率 | 加盟端利润率 | 加盟端利润（百万元） | 2026Q1 利润（百万元） | 2026Q1 利润（亿元） |
| --- | ---: | ---: | ---: | ---: |
| 11% | 16% | 1,183.1 | 461.8 | 4.62 |
| 10% | 17% | 1,257.0 | 535.7 | 5.36 |
| 9% | 18% | 1,330.9 | 609.7 | 6.10 |
| 8% | 19% | 1,404.9 | 683.6 | 6.84 |

所以当前真正最敏感的变量，不是门店数，而是：

- `10% 折扣池到底实际用了多少`
