# GMV和利润推导过程

更新时间：2026-03-30

## 1. 本文档用途

从现在开始，`霸王茶姬` 这条线只继续维护这一张主文档。

这张表只做一件事：

- 用同一条利润链，直接给出 `2025Q1 实际` 和 `2026Q1 测算` 的可对比结果

本文里的“利润”，统一指与 `2025Q1` 官方净利润尽量对齐的研究口径近似值。

## 2. 当前结论

如果严格按 [06_政策变化前后总部利润链路.md](E:/Codex/investment/霸王茶姬研究/06_政策变化前后总部利润链路.md) 的 `新模式` 去算，并且把：

- `折扣池实际使用率 = 10%`
- `总部费用率 = 11.8%`

作为基准，那么：

- `2026Q1 利润 ≈ 5.1 亿元`

和 `2025Q1` 官方净利润 `6.8 亿元` 相比：

- `同比约 -24.6%`

这版结论的关键点是：

- `GMV` 没有明显塌掉
- `直营扩张` 确实带来了利润增量
- 但 `加盟端利润` 在新模式下未必会明显抬升
- 更关键的是，当前竞争环境和品牌阶段决定了 `总部费率` 很难回到 `2025Q1` 的低位

## 3. 统一利润链

后面统一按这一条式子推：

`利润 = 加盟端利润 + 直营端门店经营利润 - 总部费用`

其中：

- `加盟端利润`：本次严格按 `新模式` 链条测算
- `直营端门店经营利润`：按 `直营GMV -> 实收 -> 原材料成本 -> 门店经营成本 -> 利润` 测算
- `总部费用`：`其他运营成本 + 销售费用 + 管理费用`

`2026Q1 平均营业门店数 = (2025Q4 期末门店数 + 2026Q1 期末门店数) / 2`

## 7. 2026Q1 利润详细推导

### 7.1 加盟端利润：严格按新模式链条

这次不再直接拍一个“加盟端利润率”，而是直接按 `文档 6` 的新模式去走：

`加盟GMV -> 17%服务费收入 -> 折扣池偏差 -> 加盟端利润`

其中：

`折扣池偏差 = 10%折扣池 - 实际折扣使用率`

基准情景下，我先取：

- `实际折扣使用率 = 10%`

也就是：

- `折扣池基本用满`
- `折扣池偏差 = 0`

因此基准情景下的加盟端利润率就是：

- `17%`

所以：

`加盟端利润 = 7,394.1 × 17% = 1,257.0`

即：

- `2026Q1 加盟端利润 = 1,257.0 百万元`

### 7.2 直营端门店经营利润

直营端继续按你前面确认的更合理口径来推：

- `直营实收率 = 80%`
- `直营毛利率 = 62%`
- `直营门店经营成本率 = 15.5%`

对应链条：

| 项目 | 2026Q1 基准（百万元） | 计算方式 |
| --- | ---: | --- |
| 直营GMV | 860.5 | 上一步测算 |
| 实收 | 688.4 | `860.5 × 80%` |
| 原材料成本 | 327.0 | `860.5 × (1 - 62%)` |
| 门店毛利收入 | 361.4 | `688.4 - 327.0` |
| 直营门店经营成本 | 133.4 | `860.5 × 15.5%` |
| 直营端门店经营利润 | 228.0 | `361.4 - 133.4` |

所以：

- `2026Q1 直营端门店经营利润 = 228.0 百万元`

### 7.3 总部费率专题判断

这一段这次单独拆开，不再直接拍一个 `11.5%` 或 `12%`。

先看历史：

| 期间 | 其他运营成本 / GMV | 销售费用 / GMV | 管理费用 / GMV | 总部费用 / GMV | 备注 |
| --- | ---: | ---: | ---: | ---: | --- |
| 2025Q1 实际 | 2.1% | 3.6% | 4.3% | 10.0% | 扩张仍快，但补贴战和海外投入还没到最重 |
| 2025Q2 实际 | 2.1% | 4.8% | 11.7% | 18.6% | 明显被 IPO 相关股份支付扭曲 |
| 2025Q2 剔除股份支付后 | 1.9% | 4.4% | 5.4% | 11.7% | 更接近真实经营强度 |
| 2025Q3 实际 | 2.3% | 3.8% | 6.5% | 12.6% | 竞争加剧，直营和海外更重 |
| 2025Q3 近似剔除股份支付后 | 2.2% | 3.7% | 5.4% | 11.3% | 更接近可持续口径 |

这里最关键的判断是：

1. `2025Q2` 的高点不能直接用  
因为官方已经明确说明，`Q2` 的利润被 `IPO 触发的股份支付` 大幅扰动

2. `2025Q1` 的 10.0% 也不能直接用  
因为当时：
- 外卖补贴竞争还没像 `Q3` 那样激烈
- 直营店和海外市场的组织负担还更轻

3. 真正可参考的，是 `Q2 剔除股份支付后` 到 `Q3 近似剔除股份支付后` 这一段  
也就是：
- `11.3% ~ 11.7%`

再结合当前竞争环境和品牌现况，这个区间还要往上再抬一点点。

### 7.4 为什么我要把总部费率基准放到 11.8%

我现在把 `2026Q1` 的总部费用率基准放在：

- `11.8%`

理由分三块。

#### 7.4.1 其他运营成本：取 2.2%

官方在 `2025Q2` 和 `2025Q3` 都明确提到：

- 这块成本上升，主要来自支持加盟网络扩张、直营网络扩张、以及海外市场渗透的人力和运营投入

而到 `2026Q1`，在你的门店假设下：

- 直营店总数已经抬到 `750` 家
- 比 `2025Q3` 的 `367` 家几乎翻倍

所以这块我不认为能回到 `Q1` 的 `2.1%` 以下。  
我取：

- `2.2%`

#### 7.4.2 销售费用：取 4.1%

这块不能只看 `Q3` 的 `3.8%`，因为官方同时也说了两件事：

- `Q3` 广告费用同比下降，部分原因是品牌投放更审慎
- 但管理层又明确表示，接下来几个季度仍会继续做 `战略性品牌投入`、推 `4.0 菜单`、拓展新消费场景

同时，行业竞争环境也没有真正轻松下来：

- 官方 `Q3` 已经明确把国内杯数下滑归因于 `外卖平台补贴竞争加剧`
- 外部行业报道也显示，`2025` 年茶饮赛道经历了高强度补贴战，品牌在 `2026` 年开始转向“单店健康和盈利质量优先”的新模型

这意味着：

- 霸王茶姬不太可能无限加大投放去跟低价品牌硬拼
- 但也不可能把销售费用压回到 `2025Q1` 的低位

它更像会维持：

- `中等偏高、但比 Q2 更克制`

所以我取：

- `4.1%`

#### 7.4.3 管理费用：取 5.5%

`Q2` 的管理费用高到 `11.7%`，显然不能直接用，因为里面有大量 `IPO 触发的股份支付`

但把股份支付剔掉以后：

- `Q2` 管理费用率约 `5.4%`
- `Q3` 近似也在 `5.4%` 左右

而官方在 `Q2` 和 `Q3` 都反复强调：

- 这块费用的核心驱动是
  - 行政和研发人员扩编
  - 办公、差旅、专业服务支出增加
  - 全球扩张和运营体量扩大

再考虑到品牌现况：

- 会员规模到 `2025Q3` 已经到 `2.22 亿`
- 品牌已经不处在“低知名度、低组织复杂度”的早期阶段
- 同时又在推海外拓展和门店模型升级

所以管理费用率我不认为会掉回 `5.0%` 以下。  
我取：

- `5.5%`

#### 7.4.4 三项合并后的基准

所以我给 `2026Q1` 的总部费用率拆项基准是：

| 项目 | 基准费率 |
| --- | ---: |
| 其他运营成本 / GMV | 2.2% |
| 销售费用 / GMV | 4.1% |
| 管理费用 / GMV | 5.5% |
| 总部费用 / GMV | 11.8% |

### 7.5 总部费用金额

在 `11.8%` 的基准下：

`总部费用 = 8,254.6 × 11.8% = 974.0`

即：

- `2026Q1 总部费用 = 974.0 百万元`

### 7.6 最终利润合成

把三段合起来：

`利润 = 1,257.0 + 228.0 - 974.0 = 511.0`

即：

- `2026Q1 利润 = 511.0 百万元`
- `约 5.1 亿元`

## 9. 折扣池使用率敏感性

因为新模式下加盟端利润直接取决于：

`17% + (10% - 实际折扣使用率)`

所以只要 `折扣池` 没有被完全用满，利润会显著上移。

在 `总部费用率固定为 11.8%` 时：

| 实际折扣使用率 | 加盟端利润率 | 加盟端利润（百万元） | 2026Q1 利润（百万元） | 2026Q1 利润（亿元） |
| --- | ---: | ---: | ---: | ---: |
| 11% | 16% | 1,183.1 | 437.1 | 4.37 |
| 10% | 17% | 1,257.0 | 511.0 | 5.11 |
| 9% | 18% | 1,330.9 | 584.9 | 5.85 |
| 8% | 19% | 1,404.9 | 658.9 | 6.59 |

所以当前真正最敏感的变量，不是门店数，而是：

- `10% 折扣池到底实际用了多少`

## 10. 总部费用率敏感性

在 `折扣池实际使用率固定为 10%` 时：

| 总部费用率 | 总部费用（百万元） | 2026Q1 利润（百万元） | 2026Q1 利润（亿元） |
| --- | ---: | ---: | ---: |
| 11.5% | 949.3 | 535.7 | 5.36 |
| 11.8% | 974.0 | 511.0 | 5.11 |
| 12.0% | 990.6 | 494.5 | 4.94 |
| 12.2% | 1,007.1 | 478.0 | 4.78 |

这张表说明：

- `总部费用率` 每上升 `0.3pct`，利润大致少 `0.25 亿`

也就是说，如果你对竞争环境更悲观，或者认为品牌会继续加大投放与组织投入，那么利润中枢还要继续往下压。

所以现在真正还值得继续补的，只剩两件事：

1. `折扣池实际使用率` 的更可信判断
2. `总部费用率` 最终更接近 `11.8%` 还是 `12.0%+`

如果后面继续更新，我就只在这一张表里把这两个变量继续收紧。
