# 研究口径与主假设

更新时间：2026-03-29

## 1. 研究目标

这轮只研究一件事：

`政策变化前后，总部利润是怎么从收入一步步走到利润的`

统一拆成三段：

1. `加盟端利润`
2. `直营端门店经营利润`
3. `减：总部费用`

也就是：

`总部利润 = 加盟端利润 + 直营端门店经营利润 - 总部费用`

## 2. 加盟端口径

### 2.1 旧模式

旧模式下，加盟端收入分两块：

1. `卖货收入`
2. `服务费收入`

所以旧模式加盟端利润率写成：

`加盟端利润率 = 卖货收入率 × 卖货毛利率 + 服务费率`

### 2.2 新模式

新模式下，加盟端不能直接按 `27%` 记利润。

当前口径统一按：

1. `17%` 是总部更接近报表确认的服务费率
2. `10%` 是折扣池
3. `27%` 更接近加盟体系重分配后的总框架，不直接等于总部利润率

所以新模式加盟端利润率写成：

`加盟端利润率 = 17% + 折扣池偏差率`

其中：

`折扣池偏差率 = 10% - 实际折扣使用率`

补充约定：

1. 若实际折扣使用率 `<=10%`，未用完部分留在总部收益里
2. 若实际折扣使用率 `>10%`，超出部分再额外冲减总部利润

### 2.3 物料与仓储口径

当前统一按你最新确认的口径：

1. 物料、仓储、履约成本已经都体现在加盟商进货价里
2. 总部在物料这里 `既不赚钱，也不亏钱`

所以这轮推导里：

`物料与仓储对总部利润净贡献近似为0`

## 3. 直营端口径

直营端统一按下面顺序，不再混用“收入率”这个词：

`直营GMV -> 实收 -> 原材料成本 -> 门店毛利收入 -> 门店经营利润`

公式统一写成：

1. `实收 = 直营GMV × 实收率`
2. `原材料成本 = 直营GMV × (1 - 毛利率)`
3. `门店毛利收入 = 实收 - 原材料成本`
4. `直营端门店经营利润 = 门店毛利收入 - 直营门店经营成本`

如果写成对直营GMV的利润率，就是：

`直营端门店经营利润率 = 实收率 - (1 - 毛利率) - 直营门店经营成本率`

其中：

`直营门店经营成本率 = 直营门店经营成本 / 直营GMV`

## 4. 总部费用口径

总部费用单独看，不先压到加盟端或直营端。

统一定义为：

`总部费用 = 其他运营成本 + 销售费用 + 管理费用`

并且分母统一用：

`总GMV`

原因只有一个：

`直营店同样占用总部销售、运营和管理资源`

## 5. 当前主假设

### 5.1 你已经给定的假设

1. `2026Q1末国内门店 = 7100`，其中直营 `500`
2. `2026Q1末海外门店 = 400`，其中直营 `250`
3. 国内店效同比 `-15%`
4. 海外店效同比 `-20%`
5. `直营实收率 = 80%`
6. `直营毛利率 = 62%`
7. 折扣池未用完部分留在总部收益

### 5.2 当前从历史数据反推的假设

1. `直营店单店月GMV / 加盟店单店月GMV ≈ 1.33x`
2. `旧模式卖货收入率 = 34.8% ~ 36.8%`
3. `旧模式卖货毛利率 = 42.8%`
4. `旧模式服务费率 = 约3%`
5. `公司层总部费用率 = 9.5% ~ 10.0%`

## 6. 当前真正要比较的两条式子

旧模式：

`总部利润率 = 卖货收入率 × 卖货毛利率 + 服务费率 + 直营端门店经营利润率 - 总部费用率`

新模式：

`总部利润率 = 17% + 折扣池偏差率 + 直营端门店经营利润率 - 总部费用率`
