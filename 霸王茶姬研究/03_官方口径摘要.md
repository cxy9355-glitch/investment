# 霸王茶姬官方口径摘要

更新时间：2026-03-29

## 1. 官方直接给了什么

### 1.1 经营数据

- 总门店数
- 加盟门店数
- 直营门店数
- 大中华区 / 海外门店数
- 大中华区 / 海外 GMV
- 大中华区单店月均 GMV
- 同店 GMV 增速

### 1.2 财务数据

- 加盟收入
- 直营收入
- 总收入
- 原材料、仓储、物流成本
- 直营门店经营成本
- 其他运营成本
- 销售费用
- 管理费用
- 经营利润
- 税前利润
- 净利润

## 2. 官方没有直接给什么

1. `国内加盟` 单独收入和利润
2. `国内直营` 单独收入和利润
3. `海外加盟` 单独收入和利润
4. `海外直营` 单独收入和利润

所以后续分部测算只能做 `受约束拆分`，不能假装官方已经给了四张利润表。

## 3. Q1 到 Q3 官方最有用的解释

### 3.1 加盟收入

- `2025Q1`：加盟收入增长主要来自加盟网络扩张
- `2025Q2`：加盟收入仍增长，但增速明显放缓
- `2025Q3`：加盟收入回落，官方归因到大中华区杯数下降和外卖平台补贴竞争加剧

### 3.2 直营收入

- `2025Q1~Q3`：直营收入增长，官方持续归因于直营门店数量增加
- `2025Q3`：明确提到大中华区和海外直营网络同时扩张

### 3.3 成本与利润率

- 供应链成本在 `2025Q2` 和 `2025Q3` 出现改善，官方归因于开店节奏放缓和采购议价改善
- 直营门店经营成本持续上升，官方归因于直营店数量增加带来的工资、租金和水电等门店费用上升
- `2025Q2` 利润率大幅走弱，核心不是 GMV 崩塌，而是总部费用显著抬升，尤其股份支付和扩编相关费用
