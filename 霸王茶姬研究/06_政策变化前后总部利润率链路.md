# 政策变化前后总部利润链路

更新时间：2026-03-29

## 1. 总体结构

`总部利润 = 加盟端利润 + 直营端门店经营利润 - 总部费用`

## 2. 加盟端链路

### 2.1 旧模式

`加盟端利润率 = 卖货收入率 × 卖货毛利率 + 服务费率`

### 2.2 新模式

`加盟端利润率 = 17% + 折扣池偏差率`

其中：

`折扣池偏差率 = 10% - 实际折扣使用率`

补充口径：

1. 若实际折扣使用率 `<=10%`，未用完部分留在总部收益
2. 若实际折扣使用率 `>10%`，超出部分额外冲减总部利润
3. 物料、仓储、履约已体现在加盟商进货价里，总部在物料这里既不赚钱也不亏钱

## 3. 直营端链路

`直营GMV -> 实收 -> 原材料成本 -> 门店毛利收入 -> 门店经营利润`

### 3.1 实收

`实收 = 直营GMV × 实收率`

### 3.2 原材料成本

`原材料成本 = 直营GMV × (1 - 毛利率)`

### 3.3 门店毛利收入

`门店毛利收入 = 实收 - 原材料成本`

### 3.4 门店经营利润

`直营端门店经营利润 = 门店毛利收入 - 直营门店经营成本`

写成利润率就是：

`直营端门店经营利润率 = 实收率 - (1 - 毛利率) - 直营门店经营成本率`

## 4. 总部费用链路

`总部费用率 = 其他运营成本率 + 销售费用率 + 管理费用率`

总部费用率统一按 `总GMV` 口径看，不直接压给加盟端。

## 5. 最终两条式子

旧模式：

`总部利润率 = 卖货收入率 × 卖货毛利率 + 服务费率 + 直营端门店经营利润率 - 总部费用率`

新模式：

`总部利润率 = 17% + 折扣池偏差率 + 直营端门店经营利润率 - 总部费用率`
