   - 总直营平均店数：`(615 + 750) / 2 = 682.5`
4. 真正应该纠正的是两点：
   - **必须把“国内直营 / 海外直营 / 总直营”同时列出来**

所以这份复核稿的目标不是把原来 `4.4 亿` 推翻掉，而是把整条链拆开，让你能判断这版到底哪里是事实，哪里是判断。

## 四、GMV 怎么拆

### 4.1 Q4 区域月均 GMV

国内区域月均 GMV：

`6,951.0M / 7,108 家 / 3 个月 = 325,970.74 元`

其中：

- `7,108 = 6,700 国内加盟 + 408 国内直营`

海外区域月均 GMV：

`371.9M / 345 家 / 3 个月 = 359,323.67 元`

其中：

- `345 = 138 海外加盟 + 207 海外直营`

### 4.2 直营 / 加盟单店效率比

这一项**不是财报直接给的**，是继承旧版模型的拆分假设。

旧版校准后：

- 国内直营店均月 GMV = `553,568`
- 国内加盟店均月 GMV = `416,216`

所以直营/加盟效率比约为：

`553,568 / 416,216 = 1.3300`

为了保持模型一致性，这版沿用：

- `直营单店月均 GMV = 加盟单店月均 GMV × 1.3300`

这一步没有官方披露直接支撑，只是一个拆分工具，所以你完全可以单独替换它。

### 4.3 基准情景下的分部月均 GMV

我在新版基准情景里设的是：

- 国内区域月均 GMV = `340,000`
- 海外区域月均 GMV = `350,000`

再结合 `1.3300` 的直营/加盟效率比，解四分部月均 GMV：

国内：

`国内加盟月均 GMV = 333,679.40`

`国内直营月均 GMV = 333,679.40 × 1.3300 = 443,794.18`

海外：

`海外加盟月均 GMV = 292,153.34`

`海外直营月均 GMV = 292,153.34 × 1.3300 = 388,564.44`

### 4.4 基准情景下的分部 GMV

公式：

`分部 GMV = 平均营业门店数 × 单店月均 GMV × 3 / 1,000,000`

逐项展开：

国内加盟：

`6,650.0 × 333,679.40 × 3 / 1,000,000 = 6,656.9M`

国内直营：

`454.0 × 443,794.18 × 3 / 1,000,000 = 604.4M`

海外加盟：

`144.0 × 292,153.34 × 3 / 1,000,000 = 126.2M`

海外直营：

`228.5 × 388,564.44 × 3 / 1,000,000 = 266.4M`

合并后：

| 项目 | GMV |
| --- | ---: |
| 加盟 GMV | `6,656.9 + 126.2 = 6,783.1M` |
| 直营 GMV | `604.4 + 266.4 = 870.8M` |
| 总 GMV | `7,653.9M` |

## 五、利润怎么推

## 5.1 基准情景参数

| 总部费用率 | 12.2% | 根据 FY2025 ex-SBC 上修后的判断 |

### 5.2 加盟端利润贡献

公式：

`加盟端利润贡献 = 加盟 GMV × 加盟端利润贡献率`

代入：

`6,783.1 × 17.0% = 1,153.1M`

### 5.3 直营端门店经营利润

公式：

`直营端门店经营利润 = 直营GMV × 80% - 直营GMV × 38% - 直营GMV × 17%`

代入：

`870.8 × 80% = 696.6M`

`870.8 × 38% = 330.9M`

`870.8 × 17% = 148.0M`

所以：

`直营端门店经营利润 = 696.6 - 330.9 - 148.0 = 217.7M`

### 5.4 总部费用

公式：

`总部费用 = 总 GMV × 总部费用率`

代入：

`7,653.9 × 12.2% = 933.8M`

### 5.5 基准情景利润

总公式：

`研究口径利润 = 加盟端利润贡献 + 直营端门店经营利润 - 总部费用`

代入：

`1,153.1 + 217.7 - 933.8 = 437.0M`

四舍五入后：

- `26Q1 调整后基准利润 = 437.1M`
- `约 4.4 亿`

## 七、这次真正纠正了什么

这次复核后，我认为真正需要纠正的是下面 4 件事。

### 7.1 纠正点一：必须同时列示国内直营、海外直营、总直营

上一版最大的问题是：

- 我写了 `国内直营 500`
- 但没有同步把 `总直营 = 750` 在同一个位置写明

这会让读者非常自然地误读成：

- `25Q4 总直营 615 -> 26Q1 总直营 500`

这是表达问题，不是公式问题，但足够严重，必须纠正。

### 7.2 纠正点二：所有“继承旧模型”的参数必须单独标记

下面这些都不是财报事实：

- `26Q1 期末门店假设`
- `直营/加盟单店效率比 = 1.3300`
- `直营实收率 = 80%`
- `原材料成本率 = 38%`
- `加盟端利润贡献率 = 16%~18%`

它们都应该单独标成“模型假设”，不能和 Q4 实际披露混在一起。

### 7.3 纠正点三：总部费用率基准确实应从 11.8% 上修

旧版继续用 `11.8%` 做基准已经偏低。  
基于 `FY2025 ex-SBC`，更合理的工作基准是：

- `12.2%`

### 7.4 纠正点四：旧版 `5.0 亿偏保守` 的提法要撤掉

在 `25Q4` 财报之后，更稳的说法应当是：

- `4.4 亿` 左右是新的工作基准
- `5.0 亿以上` 更接近乐观值
