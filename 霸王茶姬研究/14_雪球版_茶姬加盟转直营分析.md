# 霸王茶姬 Q4 加盟转直营分析：这到底是在买店，还是在回购自己的品牌价值？

最近我重点看了一下霸王茶姬 `2025Q4` 的财报，核心关注点只有一个：

`Q4` 为什么直营店突然大增、加盟店突然减少？这背后到底发生了什么？  
进一步说，茶姬是不是在高价回收加盟店，甚至有可能损害股东利益？

我把结论先放前面：

第一，`Q4` 的核心不是扩张，而是 `结构重组`。  
第二，茶姬在 `Q4` 高概率确实发生了一批国内加盟店转直营。  
第三，从现有公开数据看，我还看不到“用大额现金高价回收加盟店”的直接证据。  
第四，从第一性原理看，加盟转直营的本质，不是简单开更多直营店，而是总部在回收 `门店利润、渠道控制权、品牌溢价兑现权和用户数据控制权`。  
所以这个动作，可以部分理解为：`回购自己的品牌价值`。

## 一、Q4 到底发生了什么？

先看门店结构。

`2025Q3 -> 2025Q4`：

- 总门店：`7338 -> 7453`，只增加了 `115` 家
- 加盟店：`6971 -> 6838`，减少了 `133` 家
- 直营店：`367 -> 615`，增加了 `248` 家

如果再拆开看：

- 国内加盟：`6836 -> 6700`，减少 `136` 家
- 国内直营：`240 -> 408`，增加 `168` 家
- 海外加盟：`135 -> 138`，增加 `3` 家
- 海外直营：`127 -> 207`，增加 `80` 家

这组数一摆出来，其实已经很清楚：

- 海外那部分更像正常新开直营店，因为海外加盟店没有减少，反而略增
- 真正发生“加盟转直营”的，大概率是 `国内`

所以我认为，`Q4 国内加盟转直营门店数` 最稳的区间是：

- 下限：`136` 家
- 上限近似：`168` 家
- 中枢：`150` 家左右

换句话说，`Q4` 不是单纯多开了一堆直营店，而是茶姬主动把一批加盟店重新收回到了自己手里。

## 二、商誉为什么突然增加了 8000 多万？

这是这份财报里最值得盯的一项。

`2025Q3 -> 2025Q4`：

- 商誉：`1183.7 万 -> 9798.5 万`
- 增量：`8614.8 万`

这个变化的含义很重要。

如果只是普通新开直营店，一般会增加的是：

- 固定资产
- 使用权资产
- 装修设备
- 租赁相关资产

但一般不会“凭空长出商誉”。

所以这 `8614.8 万` 基本说明：

- 茶姬在 `Q4` 做的，不只是新开店
- 而是收购了一部分 `带经营业务的门店` 或 `经营单元`

如果按前面推测的 `136-168 家` 转直营门店数去摊，单店商誉大致是：

- `51 万 - 63 万/店`

这意味着什么？

意味着茶姬为了把这些门店收回来，确实支付了一笔溢价。  
但这笔溢价本身并不算特别离谱，更重要的是要看：`它到底花了多少现金`。

## 三、那茶姬到底花了多少钱？

这里最容易犯的错，是把 `账上资产增加` 直接理解成 `公司掏了这么多现金`。

这一步不能这么看。

比如 `Q3 -> Q4`：

- 使用权资产增加了 `2.808 亿元`
- 但租赁负债同时增加了 `2.731 亿元`

也就是说，很多“房租相关资产”其实不是公司真金白银买出来的，而是门店租约和租赁负债一起进表了。  
所以，不能把这部分直接当成现金收购价。

如果只盯真正更接近“现金流出”的部分，我做了一个保守上限测算：

先看 `Q4` 的投资现金流：

- 投资现金流净流出：`3.202 亿元`

其中还有一部分是短期投资配置：

- 新增短期投资：`1.001 亿元`

扣掉之后，剩余投资现金流大概是：

- `2.201 亿元`

然后再用 `Q2->Q3` 这段“没有商誉变化”的样本，去估正常新开直营店本来就需要占用多少投资现金流，最后剩下那部分，才更接近“加盟转直营额外消耗的现金”。

按这个思路算下来，单店额外现金流出的保守上限，大概是：

- 如果转了 `136` 家：`36.1 万/店`
- 如果转了 `150` 家：`47.0 万/店`
- 如果转了 `168` 家：`58.3 万/店`

这里要强调：

这不是在说茶姬一定只花了这么多钱。  
而是在说：`按公开报表，Q4 能被解释成转直营额外现金流出的空间，大概就在这个量级。`

所以如果你担心的是：

- 茶姬是不是用大额现金高价回收加盟店
- 是否存在明显损害股东利益的行为

那现有数据给出的信号是：

`至少目前，我还看不到“大额现金高价回收”的直接证据。`

## 四、经营现金流为正，会不会是在美化报表？

这也是一个很值得讨论的问题。

`2025Q4`：

- 经营现金流：`+2.16 亿元`
- 投资现金流：`-3.20 亿元`
- 融资现金流：`-12.01 亿元`

其中融资现金流这 `-12.01 亿元`，高概率主要就是：

- `Q3` 已经宣布
- `Q4` 实际支付

的特别分红。

真正值得分析的是前两项：

- 为什么经营现金流还是正的？
- 为什么投资现金流突然这么差？

从会计分类上讲，如果茶姬在 `Q4` 做的是：

- 收购门店业务
- 回收加盟店
- 增加长期资产和并表资产

那么这部分现金确实更应该进 `投资现金流`，而不是 `经营现金流`。

所以，仅凭现有报表，我不能直接下结论说：

`茶姬在故意错分现金流、美化报表。`

但是，如果从投资分析的角度看，这里仍然有一个容易误导人的地方：

- 如果管理层只强调“经营现金流为正”
- 却不去强调 `Q4` 里其实有一大块与加盟转直营相关的投资现金流

那投资者就会高估这家公司的“轻资产含金量”。

也就是说：

- 从会计层面，我暂时看不到明确错分证据
- 但从经济实质看，这部分投资现金流，本质上已经很接近 `主业再投资`

所以更合理的看法是：

`Q4 的经营现金流为正，不代表茶姬没为加盟转直营付出代价。`

## 五、从第一性原理看，加盟转直营的本质是什么？

我觉得这是这件事最重要的部分。

一家加盟店，本质上有四层价值：

第一层，`品牌流量`  
第二层，`门店经营利润`  
第三层，`渠道控制权`  
第四层，`用户数据和执行质量`

在加盟模式下：

- 品牌归总部
- 但门店利润的一部分归加盟商
- 门店经营决策的一部分归加盟商
- 用户数据和执行质量也会有折损

所以当品牌进入一个新阶段时，总部会面临一个问题：

`品牌越来越值钱，但总部没有把品牌价值完整地变现。`

这时候，加盟转直营的本质就很清楚了：

- 总部要把原来分给加盟商的那部分价值重新拿回来

更具体地说，茶姬回收的不是“门店外壳”，而是：

### 1. 回收利润池

加盟商原本赚走的店级利润，总部重新收回。

### 2. 回收品牌溢价的兑现权

如果品牌足够强，真正应该拿走更多利润的，不是加盟商，而是品牌本身。  
总部转直营，就是把这部分品牌溢价重新拿回自己手里。

### 3. 回收渠道控制权

当行业价格战、平台补贴战越来越激烈时，总部会越来越在意：

- 门店体验
- 定价执行
- 上新节奏
- 外卖和平台渠道协同

直营的价值就在于：总部可以直接控制。

### 4. 回收数据和组织能力

加盟模式天然有信息折损。  
而直营化，本质上是在把门店的经营数据和执行能力重新拉回总部体系。

所以，如果你问：

`茶姬是不是在回购自己的品牌价值？`

我认为答案是：

`可以这么理解，但更准确的说法是：茶姬在回购自己品牌价值在渠道端的分配权。`

## 六、这件事商业上合理吗？

如果只盯着“资产看起来有多少”，很容易误判。  
更实用的方法，是看它大概多久能回本。

在当前公开数据下，我更愿意盯的是：

- 单店额外现金流出上限：`36 万 - 58 万`

再结合 `Q4` 店效和外部加盟商利润样本做一个保守 sanity check：

- 弱店年利润：大约 `30 万`
- 中性店年利润：大约 `42 万`
- 较好店年利润：大约 `48 万`

对应的回本周期大概是：

- 现金流出 `36 万`：`0.8 - 1.2 年`
- 现金流出 `47 万`：`1.0 - 1.6 年`
- 现金流出 `58 万`：`1.2 - 1.9 年`

如果这个测算大方向不偏，那么从商业角度看：

- 茶姬回收这批门店，`是可以解释得通的`
- 至少目前看，不像是一个明显离谱、明显损害股东的交易

## 七、我的最终判断

把这件事压缩成一句话：

`2025Q4，茶姬做的不是单纯扩店，而是在通过加盟转直营，回收品牌价值在门店端的兑现权。`

现金流上：

- `经营现金流为正` 不代表这件事没花钱
- 但当前公开数据也不足以支持“公司通过高价现金回收加盟店来损害股东”的结论

本质上：

- 这不是简单买店
- 而是总部在重新分配：
  - 品牌价值
  - 门店利润
  - 渠道控制权
  - 数据和执行权

如果后续 `20-F` 或更完整附注出来，最值得继续盯的就两件事：

- 有没有 `business combination` 的详细对价分摊
- 这批被回收门店背后，是否存在 `关联方`
