# 2026Q1 利润重算

更新时间：2026-03-30

## 1. 先说结论

上一版把 `26Q1` 利润推得偏高，主要问题不在 `GMV`，而在：

1. 我把推导出来的 `直营收入 / 直营GMV` 误当成了可以直接锚定的 `实收率`
2. 我后面又把这个偏高的直营链条，接到了总部利润里
3. 总部费用率也取得偏低

重算之后，我认为更合理的 `2026Q1` 利润区间应该是：

- `保守：4.6 亿元`
- `基准：5.4 亿元`
- `偏乐观：6.4 亿元`

如果只给一个当前中心值：

- `2026Q1 利润约 5.3 亿 ~ 5.5 亿元`

## 2. 我认为上一版错在哪里

### 2.1 直营实收率不能直接从反推值硬锚

你指出得对：

- 直营门店考虑外卖占比和平台抽成后
- 实收率一般更合理的中枢就是 `75% ~ 85%`
- `80%` 作为中心值是对的

我上一版把用分部反推得到的 `84% ~ 94%` 直接拿来当硬锚，这一步不稳。

因为那个值受两个噪音影响很大：

1. `直营GMV` 本身不是官方直接披露，而是受约束拆分出来的
2. 我们当前用的是期末门店和店效比去拆，天然有误差

所以：

- `84% ~ 94%` 可以作为“提醒旧假设可能偏低”的信号
- 但不能直接拿来当 `26Q1` 的核心实收假设

### 2.2 利润推导应该尽量贴着历史实际结果走

你要的是 `26Q1 利润`，不是一个漂亮的利润率公式。

所以更稳的做法不是继续堆公式，而是：

1. 先用你给的店效变化，把 `GMV` 算出来
2. 再用历史真实利润链，去给 `加盟端利润`、`直营端门店经营利润`、`总部费用` 各自定区间

## 3. 这次重算怎么做

### 3.1 总 GMV

当前沿用 `05_推导输入数据表.md` 里的第一版 GMV：

- 加盟GMV：`7,332.6 百万元`
- 直营GMV：`987.0 百万元`
- 总GMV：`8,319.5 百万元`

### 3.2 直营端重算

这次不再用高实收率去推，而是回到你给定的中心口径：

- `直营实收率 = 80%`

再结合历史实际：

- `2025Q1` 直营收入减直营经营成本 = `85.8 百万元`
- `2025Q3` 直营收入减直营经营成本 = `125.3 百万元`

对应到直营收入的门店经营利润率，大致落在：

- `31.6% ~ 35.3%`

如果把它再乘回 `80%` 实收率，则 `直营端门店经营利润 / 直营GMV` 更合理的区间约为：

- `25% ~ 28%`

这比上一版用 `28% ~ 38%` 明显收敛，也更符合你说的实收约束。

### 3.3 加盟端重算

加盟端不再用偏乐观链条，而是尽量贴近 `2025Q1` 和 `2025Q3` 的历史残余贡献。

按历史真实利润链反推：

- `2025Q1` 加盟端对 GMV 的利润贡献大约 `17.8%`
- `2025Q3` 大约 `17.0%`

考虑到：

- 单店继续承压
- 平台补贴竞争仍在
- `2026-01-01` 新模式不一定马上把总部利润抬得更高

所以这次把加盟端利润率收敛到：

- `16.5% ~ 17.5%`

### 3.4 总部费用率重算

上一版我沿用了 `9.5% ~ 10.0%`，这偏低。

因为到 `2025Q3`：

- 直营店占比已经更高
- 海外直营扩张更快
- 总部销售、管理、运营资源的消耗已经更重

所以这次把总部费用率抬到：

- `11.0% ~ 12.0%`

这更符合 `2025Q3` 后的组织形态。

## 4. 重算结果

### 4.1 保守情景

假设：

- 加盟端利润率：`16.5%`
- 直营端门店经营利润率：`25.0%`
- 总部费用率：`12.0%`

则：

- `2026Q1 利润 ≈ 4.6 亿元`

### 4.2 基准情景

假设：

- 加盟端利润率：`16.8%`
- 直营端门店经营利润率：`26.5%`
- 总部费用率：`11.5%`

则：

- `2026Q1 利润 ≈ 5.4 亿元`

### 4.3 偏乐观情景

假设：

- 加盟端利润率：`17.5%`
- 直营端门店经营利润率：`28.0%`
- 总部费用率：`11.0%`

则：

- `2026Q1 利润 ≈ 6.4 亿元`

## 5. 为什么我现在更相信这版

因为这版的链条更贴近真实经营：

1. `直营实收率` 回到你指出的合理中心 `80%`
2. `直营利润` 不再从反推的高实收率出发，而是回到历史真实 `直营收入 - 直营经营成本`
3. `加盟端利润率` 不再往上推，而是贴着 `2025Q1` 和 `2025Q3` 的真实残余贡献
4. `总部费用率` 也抬到了更符合直营扩张阶段的位置
